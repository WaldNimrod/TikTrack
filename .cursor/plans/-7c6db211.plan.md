# תוכנית מקיפה - תיקון גרפים TradingView

## מטרה

יצירת מערכת CSS מאוחדת ומדויקת לכל גרפי TradingView Lightweight Charts במערכת, תוך עמידה קפדנית בדרישות האובייקט ויצירת תיעוד מקיף.

## עקרונות יסוד

1. **קובץ CSS נפרד** - `06-components/_tradingview-charts.css` - סגנונות ספציפיים ל-TradingView בלבד
2. **דרישות TradingView** - כל קונטיינר חייב: `display: block`, `width: 100%`, `height` מוגדר, `position: relative`, `box-sizing: border-box`
3. **אין !important** - כל הסגנונות ללא !important (חוץ ממקרים קריטיים מאושרים)
4. **RTL Support** - תמיכה מלאה ב-RTL עם logical properties
5. **צבעים דינמיים** - שימוש ב-CSS variables מהמערכת

## כללי עבודה

### דרישות TradingView Lightweight Charts:

1. **Container Element:**

                                                                                                                                                                                                - חייב להיות `display: block`
                                                                                                                                                                                                - חייב להיות עם `width: 100%` (או width מוגדר)
                                                                                                                                                                                                - חייב להיות עם `height` מוגדר (לא auto)
                                                                                                                                                                                                - חייב להיות עם `position: relative` (אם יש overlays)
                                                                                                                                                                                                - חייב להיות עם `box-sizing: border-box`
                                                                                                                                                                                                - אין להשתמש ב-flexbox על הקונטיינר עצמו

2. **Wrapper Element:**

                                                                                                                                                                                                - יכול להיות flex/grid
                                                                                                                                                                                                - חייב להיות עם `width: 100%`
                                                                                                                                                                                                - חייב להיות עם `min-width: 0` (פותר בעיות flex)
                                                                                                                                                                                                - חייב להיות עם `box-sizing: border-box`

3. **Section Layout:**

                                                                                                                                                                                                - `.section-body` חייב להיות `display: block` (לא flex)
                                                                                                                                                                                                - `.section-body` חייב להיות עם `width: 100%`
                                                                                                                                                                                                - `.section-body` חייב להיות עם `overflow: hidden`

### RTL Support:

- TradingView charts הם תמיד LTR (left-to-right)
- Wrapper יכול להיות RTL
- יש להשתמש ב-`direction: ltr` על הקונטיינר ב-RTL mode

### צבעים דינמיים:

- שימוש ב-`var(--apple-bg-primary)` לרקע
- שימוש ב-`var(--apple-border-light)` לגבולות
- שימוש ב-`var(--color-text-primary)` לטקסט

3. **תיעוד מדויק** - כל דרישה חייבת להיות מתועדת
4. **אין קיצורי דרך** - כל דרישה של TradingView חייבת להיות מיושמת
