## 🔧 **בדיקת אילוצים - הגדרה**

### מטרת בדיקת האילוצים
בדיקת אילוצי מסד נתונים וטריגרים אוטומטיים לכל עמוד במערכת, כולל בדיקת המשוב המלא בממשק המשתמש.

### תהליך בדיקת האילוצים - שלב א: הגדרת אילוצים
1. **בדיקת אילוצי ENUM** - וידוא שהסטטוסים הנכונים מוגדרים
2. **בדיקת טריגרים אוטומטיים** - וידוא שטריגרים מעדכנים שדות אוטומטית
3. **בדיקת אילוצי NOT NULL** - וידוא שדות חובה מוגדרים נכון
4. **בדיקת אילוצי Foreign Key** - וידוא קשרים בין טבלאות
5. **בדיקת אילוצי UNIQUE** - וידוא שדות ייחודיים
6. **בדיקת אילוצי RANGE** - וידוא טווחי ערכים תקינים

### תהליך בדיקת האילוצים - שלב ב: אכיפה ברמת השרת
1. **בדיקת ValidationService** - וידוא שהאילוצים נאכפים ב-API
2. **בדיקת הודעות שגיאה** - וידוא שהודעות שגיאה ברורות מוחזרות
3. **בדיקת קוד סטטוס** - וידוא שמוחזר קוד שגיאה 400 עבור נתונים לא תקינים
4. **בדיקת מבנה JSON** - וידוא שמבנה התגובה תקין

### תהליך בדיקת האילוצים - שלב ג: משוב בממשק המשתמש
1. **בדיקת טיפול בשגיאות** - וידוא שצד הלקוח מטפל בשגיאות מהשרת
2. **בדיקת מערכת ההתראות** - וידוא שימוש ב-`showValidationWarning` ו-`showErrorNotification`
3. **בדיקת הודעות בעברית** - וידוא שהודעות מוצגות בעברית ברורה
4. **בדיקת זיהוי שדות** - וידוא שכל שגיאה מקושרת לשדה הספציפי
5. **בדיקת הצגת שגיאות מרובות** - וידוא שכל השגיאות מוצגות בנפרד

### דוגמאות לבדיקות אילוצים
- **חשבונות**: סטטוסים `('open', 'closed', 'cancelled')`
- **טיקרים**: סטטוסים + טריגרים לעדכון `active_trades` ו-`status`
- **טריידים**: סטטוסים + אילוצי `account_id` ו-`ticker_id`
- **תכניות טרייד**: סטטוסים + אילוצי `investment_type`
- **תזרימי מזומנים**: ENUM ל-`type` ו-`source`, RANGE ל-`amount` ו-`usd_rate`

### דוגמאות למשוב בממשק
- **שגיאת ENUM**: "סוג תזרים לא תקין - יש לבחור ערך מהרשימה"
- **שגיאת RANGE**: "סכום חייב להיות שונה מ-0"
- **שגיאת Foreign Key**: "חשבון לא קיים במערכת"
- **שגיאה כללית**: "שגיאה בשמירת תזרים מזומנים - בדוק את הנתונים שהוזנו"

כל עמוד עם אילוצי מסד נתונים תקינים, אכיפה ברמת השרת, ומשוב ברור בממשק המשתמש.

#### **1. בעיות מערכת התראות - צבעים שגויים**
**תיאור הבעיה**: 
- הודעות הצלחה מופיעות בכחול במקום ירוק
- הודעות שגיאה מופיעות בכחול במקום אדום

**סיבות אפשריות**:
- הפונקציה `showNotification` מקבלת פרמטרים שגויים
- קריאה רקורסיבית ב-`showValidationWarning`
- פונקציות לא מיוצאת נכון ל-global scope

**פתרון עם פונקציות כלליות**:
```javascript
// ✅ נכון - שימוש בפונקציות הכלליות
window.showSuccessNotification('הצלחה', 'עסקה נשמרה בהצלחה');
window.showErrorNotification('שגיאה', 'יש לתקן את השגיאות בטופס');
window.showValidationWarning('fieldId', 'הודעת שגיאה מפורטת');

// ❌ שגוי - קריאה רקורסיבית או פרמטרים שגויים
window.showValidationWarning('fieldId', 'message'); // רקורסיבי
window.showErrorNotification('הודעת שגיאה'); // חסר title
```

#### **2. בעיות קריאה רקורסיבית**
**תיאור הבעיה**: 
- פונקציה קוראת לעצמה וגורמת ללולאה אינסופית
- `showValidationWarning` ב-`ui-utils.js` קוראת ל-`window.showValidationWarning`

**פתרון**:
```javascript
// ✅ נכון - שימוש בפונקציה הכללית מ-notification-system.js
function showValidationWarning(field, message) {
    if (typeof window.notificationSystem !== 'undefined' && window.notificationSystem.showValidationWarning) {
        window.notificationSystem.showValidationWarning(field, message);
    } else if (window.showErrorNotification) {
        window.showErrorNotification('שגיאת וולידציה', message);
    }
}

// ❌ שגוי - קריאה רקורסיבית
function showValidationWarning(field, message) {
    if (typeof window.showValidationWarning === 'function') {
        window.showValidationWarning(field, message); // רקורסיבי!
    }
}
```

#### **3. בעיות ייצוא פונקציות**
**תיאור הבעיה**: 
- פונקציות לא זמינות ב-global scope
- התנגשויות בין פונקציות עם אותו שם

**פתרון**:
```javascript
// ✅ נכון - ייצוא נכון ב-notification-system.js
window.showValidationWarning = showValidationWarning;
window.showErrorNotification = showErrorNotification;
window.showSuccessNotification = showSuccessNotification;

// ✅ נכון - ייצוא כמודול
window.notificationSystem = {
    showValidationWarning,
    showErrorNotification,
    showSuccessNotification
};

// ❌ שגוי - ייצוא כפול או התנגשות
window.showValidationWarning = showValidationWarning; // ב-ui-utils.js
window.showValidationWarning = showValidationWarning; // ב-notification-system.js
```

#### **5. טיפול בשגיאות ולידציה מהשרת**
```javascript
// ✅ נכון - טיפול מפורט בשגיאות מהשרת
const result = await response.json();

if (response.ok && result.status === 'success') {
  window.showSuccessNotification('הצלחה', 'הפריט נשמר בהצלחה!');
} else {
  // טיפול בשגיאות וולידציה מהשרת
  if (result.error && result.error.message) {
    const serverMessage = result.error.message;
    
    if (serverMessage.includes('validation failed')) {
      const validationErrors = serverMessage.replace('Validation failed: ', '').split('; ');
      
      validationErrors.forEach(error => {
        let fieldError = error;
        let fieldName = '';
        
        // תרגום שגיאות ספציפיות
        if (error.includes("Field 'name' is required")) {
          fieldError = 'שם הוא שדה חובה';
          fieldName = 'nameError';
        }
        
        // שימוש במערכת ההתראות המובנת
        if (fieldName && window.showValidationWarning) {
          window.showValidationWarning(fieldName, fieldError);
        } else {
          window.showErrorNotification('שגיאת וולידציה', fieldError);
        }
      });
    } else {
      window.showErrorNotification('שגיאה', serverMessage);
    }
  }
}
```

### **🔧 פונקציות זמינות**

#### **1. פונקציות התראות בסיסיות**

##### **`window.showNotification(message, type, title, duration)`**
**תיאור**: פונקציה בסיסית להצגת התראה
**פרמטרים**:
- `message` (string) - תוכן ההודעה
- `type` (string) - סוג ההתראה: 'success', 'error', 'warning', 'info'
- `title` (string) - כותרת ההתראה (ברירת מחדל: 'התראה')
- `duration` (number) - משך הצגה במילישניות (ברירת מחדל: 4000)

**דוגמה**:
```javascript
window.showNotification('הנתונים נשמרו בהצלחה', 'success', 'הצלחה', 5000);
```

##### **`window.showSuccessNotification(title, message, duration)`**
**תיאור**: התראה ירוקה להצלחה
**פרמטרים**:
- `title` (string) - כותרת ההתראה
- `message` (string) - תוכן ההודעה
- `duration` (number) - משך הצגה (ברירת מחדל: 4000)

**דוגמה**:
```javascript
window.showSuccessNotification('הצלחה', 'מטבע נוסף בהצלחה למערכת');
```

##### **`window.showErrorNotification(title, message, duration)`**
**תיאור**: התראה אדומה לשגיאה
**פרמטרים**:
- `title` (string) - כותרת ההתראה
- `message` (string) - תוכן ההודעה
- `duration` (number) - משך הצגה (ברירת מחדל: 6000)

**דוגמה**:
```javascript
window.showErrorNotification('שגיאה', 'שגיאה בשמירת המטבע');
```

##### **`window.showWarningNotification(title, message, duration)`**
**תיאור**: התראה כתומה לאזהרה
**פרמטרים**:
- `title` (string) - כותרת ההתראה
- `message` (string) - תוכן ההודעה
- `duration` (number) - משך הצגה (ברירת מחדל: 5000)

**דוגמה**:
```javascript
window.showWarningNotification('אזהרה', 'המטבע כבר קיים במערכת');
```

##### **`window.showInfoNotification(title, message, duration)`**
**תיאור**: התראה כחולה למידע
**פרמטרים**:
- `title` (string) - כותרת ההתראה
- `message` (string) - תוכן ההודעה
- `duration` (number) - משך הצגה (ברירת מחדל: 4000)

**דוגמה**:
```javascript
window.showInfoNotification('מידע', 'המערכת מתעדכנת...');
```

#### **2. פונקציות וולידציה**

##### **`window.showValidationWarning(fieldId, message, duration)`**
**תיאור**: התראה אדומה עם סימון שדה ספציפי
**פרמטרים**:
- `fieldId` (string) - מזהה השדה הבעייתי
- `message` (string) - הודעת השגיאה
- `duration` (number) - משך הצגה (ברירת מחדל: 6000)

**תכונות**:
- מציג התראה אדומה
- מסמן את השדה הבעייתי באדום
- מגליל לשדה הבעייתי
- מסיר את הסימון אחרי 3 שניות

**דוגמה**:
```javascript
window.showValidationWarning('currencySymbol', 'סמל מטבע הוא שדה חובה');
```

#### **3. פונקציות אישור**

##### **`window.showDeleteWarning(itemType, itemName, itemTypeDisplay, onConfirm, onCancel)`**
**תיאור**: חלון אישור למחיקה
**פרמטרים**:
- `itemType` (string) - סוג הפריט
- `itemName` (string) - שם הפריט
- `itemTypeDisplay` (string) - שם תצוגה של סוג הפריט
- `onConfirm` (function) - פונקציה לביצוע אם מאושר
- `onCancel` (function) - פונקציה לביצוע אם בוטל

**דוגמה**:
```javascript
window.showDeleteWarning('currency', 'USD', 'מטבע',
```
