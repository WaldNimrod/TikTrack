# 🚀 הוראות הפעלת שרת SimpleTrade

## 📋 דרישות מקדימות
- Python 3.9+
- pip
- בסיס נתונים SQLite

## 🔧 התקנה
```bash
# התקנת תלויות
pip install -r requirements.txt
```

## 🎯 דרכי הפעלה

### 1. הפעלה רגילה (מומלץ לפיתוח)
```bash
python3 app.py
```

### 2. הפעלה יציבה (מומלץ לשימוש)
```bash
python3 run_server.py
```

### 3. הפעלה עם Gunicorn (מומלץ לייצור)
```bash
python3 run_production.py
```

## ⚙️ הגדרות השרת

### הגדרות יציבות:
- **Debug Mode**: כבוי (מונע רילוד אוטומטי)
- **Host**: 127.0.0.1
- **Port**: 5002
- **Threading**: מופעל
- **Reloader**: כבוי

### הגדרות בסיס נתונים:
- **Timeout**: 20 שניות
- **WAL Mode**: מופעל
- **Cache Size**: 10MB
- **Temp Store**: Memory

## 🛠️ פתרון בעיות

### השרת נעצר:
1. בדוק שהפורט 5002 פנוי
2. בדוק שיש הרשאות לקובץ בסיס הנתונים
3. נסה הפעלה עם `run_server.py`

### שגיאות חיבור:
1. בדוק שהשרת רץ
2. בדוק את כתובת ה-URL
3. בדוק את הגדרות ה-CORS

### בעיות ביצועים:
1. השתמש ב-Gunicorn
2. הגדל את מספר ה-workers
3. בדוק את גודל בסיס הנתונים

## 📊 מוניטורינג

### בדיקת סטטוס:
```bash
curl http://127.0.0.1:5002/api/stats
```

### בדיקת חיבור:
```bash
curl http://127.0.0.1:5002/
```
