## קבצים זמינים:

### ✅ **run_flask_simple.py** - השרת הפשוט (מומלץ לפיתוח)
- Flask development server עם הגדרות יציבות
- `debug=True`, `use_reloader=False` - יציבות משופרת
- `host='127.0.0.1'`, `port=8080` - הגדרות ברורות
- תמיכה בנתיבים ללא .html
- הודעות notification system
- מתאים לפיתוח יומיומי

**הפעלה:**
```bash
cd Backend
python3 run_flask_simple.py
```

### ✅ **app.py** - השרת הראשי (הפעלה ישירה)
- קובץ השרת הראשי עם כל ה-API endpoints
- מאפשר הפעלה ישירה עם Flask development server
- כולל את כל הפונקציונליות של המערכת
- מתאים להפעלה ישירה או דרך run_flask_simple.py

**הפעלה:**
```bash
cd Backend
python3 app.py
```

## 🎯 **הודעות משתמש:**

### **Notification System (חדש):**
- **מיקום:** פינה ימנית עליונה
- **זמן הצגה:** 3 שניות אוטומטית
- **סוגים:** success (ירוק), error (אדום), warning (צהוב), info (כחול)
- **יתרונות:** לא חוסם את המסך, נעלם אוטומטית, נראה מקצועי

### **דף דמו:**
- **כתובת:** `http://127.0.0.1:8080/notification-demo`
- **תכולה:** השוואה בין alert ו-notification
- **מטרה:** הדגמת ההבדל בין השיטות

### **המרה מ-alert ל-notification:**
```javascript
// ישן (לא מומלץ)
alert('הערה נשמרה בהצלחה!');

// חדש (מומלץ)
if (typeof window.showNotification === 'function') {
  window.showNotification('הערה נשמרה בהצלחה!', 'success');
} else {
  alert('הערה נשמרה בהצלחה!');
}
```
