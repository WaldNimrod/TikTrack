# תוכנית בדיקות CRUD מקיפה - TikTrack

## 📋 סקירה כללית

תוכנית זו מכסה בדיקות CRUD (Create, Read, Update, Delete) מלאות לכל העמודים במערכת TikTrack, כולל בדיקת כל הכפתורים, טפסים ופונקציות.

## 🎯 מטרות הבדיקה

1. **בדיקת תקינות CRUD** בכל העמודים העסקיים
2. **אימות כל הכפתורים** בכל עמוד
3. **בדיקת טפסים** והגשה
4. **בדיקת מיון ופילטרים** בטבלאות
5. **בדיקת מודלים** וחלונות קופצים
6. **בדיקת ניווט** וקישורים
7. **בדיקת הרשאות** ואימות נתונים

## 🧪 סוגי בדיקות לכל עמוד

### 1. בדיקות Create (יצירה)
- [ ] כפתור "הוסף חדש" עובד
- [ ] מודל הוספה נפתח
- [ ] טופס הוספה מוצג נכון
- [ ] ולידציית שדות עובדת
- [ ] שמירה מצליחה
- [ ] רשומה חדשה מופיעה בטבלה

### 2. בדיקות Read (קריאה)
- [ ] טבלת נתונים נטענת
- [ ] מיון עמודות עובד
- [ ] פילטרים עובדים
- [ ] דפדפן עובד (pagination)
- [ ] חיפוש עובד
- [ ] רענון נתונים עובד

### 3. בדיקות Update (עדכון)
- [ ] כפתור "ערוך" עובד
- [ ] מודל עריכה נפתח עם נתונים
- [ ] עריכת שדות עובדת
- [ ] שמירת שינויים מצליחה
- [ ] שינויים מופיעים בטבלה

### 4. בדיקות Delete (מחיקה)
- [ ] כפתור "מחק" עובד
- [ ] אישור מחיקה מופיע
- [ ] מחיקה מבוצעת
- [ ] רשומה נעלמת מהטבלה

### 5. בדיקות ממשק משתמש
- [ ] כל הכפתורים לחיצים
- [ ] הודעות שגיאה/הצלחה מופיעות
- [ ] טעינה מתבצעת נכון
- [ ] ניווט עובד
- [ ] RTL ועברית נכונים

### 6. בדיקות פונקציונליות מתקדמות
- [ ] ייצוא נתונים
- [ ] ייבוא נתונים
- [ ] קישורים בין ישויות
- [ ] מערכת התראות
- [ ] הרשאות משתמש
