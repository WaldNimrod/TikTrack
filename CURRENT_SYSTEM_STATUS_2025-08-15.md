# TikTrack - מצב מערכת נוכחי (2025)

## 📋 סקירה כללית

מערכת TikTrack נמצאת במצב יציב ופעיל עם שרת יציב על פורט 5002. כל הדפים פועלים כראוי עם עיצוב אחיד ומערכת טבלאות מתקדמת.

## 🚀 שרת יציב

### פרטי השרת
- **קובץ:** `Backend/app.py`
- **פורט:** 5002
- **סטטוס:** יציב ופעיל
- **הפעלה:** `./start_server.sh`

### ⚠️ חשוב!
- **אין לשנות את השרת!** - המערכת פועלת על `app.py` ולא על `app_new.py`
- **אין לשנות את הפורט!** - המערכת פועלת על פורט 5002
- **אין לשנות את מבנה הטבלאות!** - כל הטבלאות פועלות כראוי

## 🎯 דפים פעילים

### דפים עיקריים
1. **דף הבית** (`/`) - דף כניסה ראשי
2. **תכנון טריידים** (`/planning`) - טבלת תכנונים מלאה עם כפתורי פעולות
3. **בסיס נתונים** (`/database`) - תצוגת כל הטבלאות עם כפתורי פעולות
4. **מעקב טריידים** (`/tracking`) - מעקב אחר טריידים פעילים
5. **חשבונות** (`/accounts`) - ניהול חשבונות
6. **התראות** (`/alerts`) - מערכת התראות

### דפים נוספים
- **מחקר** (`/research`) - כלי מחקר
- **העדפות** (`/preferences`) - הגדרות מערכת

## 📊 בסיס נתונים

### טבלאות פעילות (עם נתונים)
- `accounts` - חשבונות משתמשים
- `tickers` - מניות ונכסים
- `trade_plans` - תכנוני טריידים
- `trades` - טריידים פעילים וסגורים
- `executions` - ביצועי טריידים
- `alerts` - התראות
- `cash_flows` - תזרים מזומנים
- `open_execution_requests` - פקודות ביצוע
- `performance_snapshots` - תמונת זמן
- `notes` - הערות

### טבלאות עתידיות (ללא נתונים כרגע)
- `users` - משתמשים (תצוגה בלבד)
- `user_roles` - תפקידי משתמשים (תצוגה בלבד)

## 🎨 עיצוב וממשק

### רכיבים מרכזיים
1. **App Header Web Component** (`app-header.js`)
   - תפריט ראשי
   - פילטרים מתקדמים
   - עיצוב אחיד

2. **Grid Table System** (`grid-table.css`)
   - עיצוב טבלאות אחיד
   - כפתורי פעולות (עריכה/מחיקה)
   - אנימציות hover

### סגנונות
- **Apple Theme** - עיצוב מודרני
- **Responsive Design** - תמיכה במכשירים שונים
- **RTL Support** - תמיכה בעברית
