# מטריצת מערכות כלליות - קובץ עבודה

## מטרת המטריצה
המטריצה נועדה להעריך את יישום המערכות הכלליות בכל עמוד במערכת, לזהות כפילויות, מערכות חסרות, ובעיות ארכיטקטורה.

## מבנה המטריצה
- **שורות:** מערכות כלליות (מסודרות לפי חשיבות)
- **עמודות:** עמודים במערכת (18 עמודים)
- **עמודת דוקומנטציה:** ציון 1-5 לרמת הדוקומנטציה של כל מערכת

## מקרא צבעים (רמזור)

### סטטוסי יישום:
- 🟢 **ירוק** - מערכת כללית מיושמת במלואה, כולל כל הקריאות אליה
- 🔵 **כחול** - מערכת כללית מיושמת אבל לא מלא או לא מדויק
- ⚫ **אפור** - אין צורך במערכת הכללית ואין מערכת דומה מקומית
- 🟠 **כתום** - יש מערכת מקומית בלבד (אין מערכת כללית)
- 🔴 **אדום** - יש מערכת כפולה מתנגשת (גם כללית וגם מקומית)
- 🟡 **צהוב** - משהו אחר/לא ברור/לא מובן

### סטטוסים נוספים:
- 🟣 **סגול** - מערכת מיושמת אבל לא נטענת/לא פעילה
- 🔵 **כחול בהיר** - מערכת מיושמת אבל יש בעיות ביצועים או שגיאות
- ⚪ **לבן** - מערכת קיימת אבל לא בשימוש (dead code)

#### 3.2 תהליך תיקון מפורט

##### 3.2.1 יצירת פונקציה כללית (אם חסרה)
```javascript
// דוגמה: יצירת toggleSection כללית
function toggleSection(sectionId) {
    const section = document.getElementById(sectionId);
    if (!section) {
        console.warn(`Section ${sectionId} not found`);
        return;
    }
    
    const sectionBody = section.querySelector('.section-body');
    const toggleBtn = section.querySelector('.toggle-btn');
    const icon = toggleBtn ? toggleBtn.querySelector('.toggle-icon') : null;
    
    if (sectionBody) {
        const isCollapsed = sectionBody.style.display === 'none';
        sectionBody.style.display = isCollapsed ? 'block' : 'none';
        
        if (icon) {
            icon.textContent = isCollapsed ? '▼' : '▶';
        }
        
        // שמירת מצב
        localStorage.setItem(`${sectionId}_collapsed`, (!isCollapsed).toString());
    }
}
```

##### 3.2.2 החלפת פונקציות מקומיות
```javascript
// לפני התיקון - פונקציה מקומית
function toggleSection(sectionId) {
    // קוד מקומי מורכב...
}

// אחרי התיקון - קריאה לפונקציה כללית
function toggleSection(sectionId) {
    if (typeof window.toggleSection === 'function') {
        window.toggleSection(sectionId);
    } else {
        console.warn('toggleSection function not found');
    }
}
```

##### 3.2.3 בדיקת קריאות
- בדיקה שכל הקריאות לפונקציה עובדות
- בדיקה שהפונקציה הכללית נטענת
- בדיקה שהפונקציונליות זהה

##### 3.2.4 עדכון HTML
```html
<!-- לפני התיקון -->
<button onclick="toggleSection('my-section')">Toggle</button>

<!-- אחרי התיקון - אם נדרש -->
<button onclick="window.toggleSection('my-section')">Toggle</button>
```

#### 3.3 כללי תיקון חשובים
1. **שמירה על פונקציונליות:** הפונקציה הכללית חייבת לעבוד בדיוק כמו המקומית
2. **שמירה על פרמטרים:** כל הפרמטרים חייבים להיות זהים
3. **שמירה על localStorage:** אם הפונקציה המקומית שמרה מצב, הפונקציה הכללית חייבת גם
4. **שמירה על console.log:** אם הפונקציה המקומית הדפיסה הודעות, הפונקציה הכללית חייבת גם
5. **טיפול בשגיאות:** הפונקציה הכללית חייבת לטפל בשגיאות כמו המקומית

### **📝 הערות חשובות:**
- עמודת "דוק' (1-5)" מציינת את רמת הדוקומנטציה של כל מערכת (1 = חסר לחלוטין, 5 = מעולה)
- כל תא במטריצה צריך להיות מסומן בצבע המתאים לפי המקרא
- יש לעבוד מערכת מערכת ולא עמוד עמוד
- לכל מערכת יש לקרוא תחילה את הדוקומנטציה שלה

### **🆕 מערכות חדשות שנוספו:**
1. **Unified IndexedDB System** - מערכת אחסון מקומי מאוחדת ומתקדמת
   - `UnifiedIndexedDB` - מתאם מרכזי לכל מערכות האחסון
   - `saveFileMapping` - שמירת מיפוי קבצים
   - `saveNotificationHistory` - שמירת היסטוריית התראות
   - `saveHeaderState` - שמירת מצב כותרת ופילטרים
   - `saveLinterScanningResults` - שמירת תוצאות סריקת Linter
   - **מיגרציה הושלמה:** File Mapping, Notification, Header, Linter Monitor

2. **File Mapping System** - מערכת מיפוי קבצי הפרויקט (מיגרציה הושלמה)
   - `discoverProjectFiles` - גילוי קבצי הפרויקט
   - `saveFileMapping` - שמירת מיפוי קבצים (עכשיו דרך UnifiedIndexedDB)
   - `getFileMapping` - קבלת מיפוי קבצים (עכשיו דרך UnifiedIndexedDB)
   - `updateFileMappingStatus` - עדכון סטטוס מיפוי

3. **Cache Management System** - מערכת ניהול מטמון
   - `clearAllCache` - ניקוי מטמון מלא
   - `clearDevelopmentCache` - ניקוי מטמון פיתוח
   - `clearCacheBeforeCRUD` - ניקוי מטמון לפני פעולות CRUD
   - `clearExpiredCache` - ניקוי מטמון פג תוקף
