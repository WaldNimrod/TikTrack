## **🚀 התחלה מהירה (היום)**

### **צעד ראשון - יצירת package.json:**
```bash
npm init -y
npm install --save-dev eslint prettier htmlhint stylelint
npx eslint --init
```

### **צעד שני - בדיקה ראשונית:**
```bash
npx eslint trading-ui/scripts/*.js
npx prettier --check trading-ui/scripts/*.js
npx htmlhint trading-ui/*.html
npx stylelint "trading-ui/styles/*.css"
```
