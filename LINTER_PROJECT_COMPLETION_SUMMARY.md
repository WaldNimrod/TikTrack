## 🔧 פקודות מלאות למערכת

### **בדיקות וניתוח:**
```bash
npm run lint                    # בדיקה בסיסית
npm run lint:fix               # תיקון אוטומטי
npm run quality:report         # דוח HTML מפורט  
npm run validate:all           # בדיקה מקיפה (JS+CSS+HTML)
npm run check:all              # בדיקת איכות שלמה
```

### **מוניטור וניהול:**
```bash
npm run watch:linter           # מעקב אחר שינויים
npm run monitor:realtime       # מוניטור + שרת יחד
npm run lint:dashboard         # פתיחת דשבורדים
scripts/linter-lifecycle-manager.sh status    # סטטוס מהיר
scripts/linter-lifecycle-manager.sh monitor   # הפעלת מוניטור מלא
```

### **אוטומציה ותחזוקה:**
```bash
scripts/daily-linter-check.sh   # בדיקה יומית
npm run lint:scheduled         # למשימות מתוזמנות
scripts/test-automation.sh     # בדיקת כל האוטומציה
npm run precommit:test         # בדיקת pre-commit hook
```
