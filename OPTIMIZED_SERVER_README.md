# TikTrack Optimized Server for Laptops

## 🎯 מטרה
שרת Flask מותאם במיוחד למחשבים ניידים עם אופטימיזציות זיכרון מתקדמות.

## ⚡ תכונות עיקריות

### 🧠 אופטימיזציות זיכרון
- **Garbage Collection אגרסיבי** - ניקוי זיכרון אוטומטי
- **לוגים מופחתים** - פחות כתיבה לדיסק
- **תהליך יחיד** - מניעת דליפות זיכרון
- **ניטור זיכרון בזמן אמת** - מעקב אחר שימוש זיכרון
- **ניקוי אוטומטי** - הסרת קבצים זמניים

### 🚀 ביצועים משופרים
- **תגובות מהירות** - זמן תגובה מופחת
- **חיבורים מוגבלים** - מניעת עומס
- **Caching חכם** - קבצים סטטיים במטמון
- **Connection pooling** - ניהול חיבורים יעיל

## 🔧 התקנה והפעלה

### 1. התקנת חבילות נדרשות
```bash
pip3 install psutil flask-cors
```

### 2. הפעלת השרת הממוטב
```bash

```

### 3. בדיקת סטטוס
```bash
./start_optimized.sh status
```

## 🛠️ פקודות זמינות

| פקודה | תיאור |
|--------|--------|
| `./start_optimized.sh` | הפעלת שרת ממוטב |
| `./start_optimized.sh stop` | עצירת השרת |
| `./start_optimized.sh restart` | הפעלה מחדש |
| `./start_optimized.sh status` | בדיקת סטטוס |
| `./start_optimized.sh clean` | ניקוי זיכרון |
| `./start_optimized.sh logs` | הצגת לוגים |
| `./start_optimized.sh errors` | הצגת שגיאות |

## 📈 השוואה עם השרת הרגיל

| מאפיין | שרת רגיל | שרת ממוטב |
|---------|-----------|------------|
| **זיכרון התחלתי** | ~50MB | ~30MB |
| **זיכרון מקסימלי** | ~200MB+ | ~100MB |
| **לוגים** | DEBUG מלא | WARNING בלבד |
| **Garbage Collection** | אוטומטי | אגרסיבי |
| **תהליכים** | מרובים | יחיד |
| **ניטור זיכרון** | לא | כן |
| **ניקוי אוטומטי** | לא | כן |

## 🔧 התאמות נוספות

### הגדרת זיכרון מקסימלי
עריכת `Backend/dev_server_optimized.py`:
```python
# שינוי סף הזיכרון (בשורה 150)
if memory_info['current_mb'] > 100:  # שנה ל-50 או 75
    memory_monitor.force_gc()
```

### הגדרת תדירות ניקוי
```python
# שינוי תדירות ניקוי (בשורה 200)
if app.request_count % 10 == 0:  # שנה ל-5 או 15
    memory_monitor.force_gc()
```
