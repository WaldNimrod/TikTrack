## 🛠️ **פקודות חשובות:**

### **אתחול שרת:**
```bash
./restart                 # זיהוי אוטומטי חכם
./restart quick          # אתחול מהיר (5-10 שניות)
./restart complete       # אתחול מלא (30-60 שניות)
./restart --interactive  # מצב אינטראקטיבי
./restart --status       # מצב המערכת
./restart --help         # עזרה
```

### **בדיקת וולידציה:**
```bash
# בדיקת וולידציה בזמן אמת
curl -X POST http://localhost:8080/api/v1/trades/ \
  -H "Content-Type: application/json" \
  -d '{"account_id": 999, "ticker_id": 1, "trade_plan_id": 1, "investment_type": "swing", "side": "Long", "status": "open"}'

# בדיקת לוגים
tail -f Backend/logs/app.log | grep ValidationService
```

### **בדיקת אילוצים:**
```bash
# צפייה בכל האילוצים
curl http://localhost:8080/api/v1/constraints/

# בדיקת וולידציה
curl -X POST http://localhost:8080/api/v1/constraints/validate \
  -H "Content-Type: application/json" \
  -d '{"table_name": "trades", "data": {...}}'
```
