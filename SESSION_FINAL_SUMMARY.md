## 🔧 **מה תיקנו - תיקון ARIA:**

### **הבעיה:**
```
Blocked aria-hidden on an element because its descendant retained focus
```
Bootstrap הוסיף `aria-hidden="true"` למודלים תוך כדי ניסיון לפקוס על אלמנט בתוכם.

### **הפתרון:**
1. **מודלים דינמיים:** `window.createAndShowModal()` + `MutationObserver`
2. **מודלים סטטיים:** הסרת `aria-hidden="true"` מה-HTML הראשוני
