## §3 FIX-1: Priority-Based Refresh Filter (Layer 1)

### §3.1 Purpose

Replace the unconditional "fetch all active tickers every cycle" loop with a priority-aware filter. Reduces API calls from N (all active tickers) to a small subset per cycle.

### §3.2 New Priority Categories

| Category | Condition | Action |
|---|---|---|
| FIRST_FETCH | `last_fetched_at IS NULL` (no price data exists for this ticker) | **Always fetch** regardless of market hours or trade status |
| HIGH | ticker has an ACTIVE trade (`user_data.trades.status = 'ACTIVE'`) AND market is currently open | **Always fetch** |
| LOW | all other cases | Fetch ONLY if `last_fetched_at` is older than `off_hours_interval_minutes` ago |

**Market-open definition:** Reuse `get_current_cadence_minutes()` from `market_data_settings.py`. If current cadence == `get_intraday_interval_minutes()`, market is open. If cadence == `get_off_hours_interval_minutes()`, market is closed.
→ Exact import: `from api.integrations.market_data.market_data_settings import get_current_cadence_minutes, get_intraday_interval_minutes`

**Active trade query (exact SQL):**
```sql
SELECT DISTINCT ticker_id FROM user_data.trades WHERE status = 'ACTIVE'
```
Note: `status` field is `VARCHAR`, values are uppercase strings. `'ACTIVE'` = open position.

**Last fetch query (exact SQL):**
```sql
SELECT fetched_at FROM market_data.ticker_prices_intraday
WHERE ticker_id = :ticker_id
ORDER BY fetched_at DESC
LIMIT 1
```

```python
async def _get_active_trade_ticker_ids(db: AsyncSession) -> set:
    """Return set of ticker_ids that have at least one ACTIVE trade."""
    result = await db.execute(
        text("SELECT DISTINCT ticker_id FROM user_data.trades WHERE status = 'ACTIVE'")
    )
    return {row[0] for row in result.fetchall()}


async def _get_last_fetched_at(db: AsyncSession, ticker_id: UUID) -> Optional[datetime]:
    """Return the most recent fetched_at for a ticker from ticker_prices_intraday, or None."""
    result = await db.execute(
        text("""
            SELECT fetched_at FROM market_data.ticker_prices_intraday
            WHERE ticker_id = :tid
            ORDER BY fetched_at DESC LIMIT 1
        """),
        {"tid": str(ticker_id)},
    )
    row = result.fetchone()
    if row and row[0]:
        ts = row[0]
        if getattr(ts, "tzinfo", None) is None:
            ts = ts.replace(tzinfo=timezone.utc)
        return ts
    return None
```

### §4.4 New Code — `_fetch_prices_batch_sync()` + `YahooProvider.get_ticker_prices_batch()`

**File:** `api/integrations/market_data/providers/yahoo_provider.py`

**Implementation pattern:** identical to all existing sync functions in this file (e.g., `_fetch_market_status_sync()` which already calls v7/finance/quote). `YahooProvider` does NOT have `_make_request` or `_base_url` internals — the correct pattern is a module-level synchronous function using `httpx.Client` wrapped by an async class method via `loop.run_in_executor`.

**Step 1 — Add module-level sync function** (add alongside `_fetch_market_status_sync`, `_fetch_price_sync`, etc.):

```python
def _fetch_prices_batch_sync(symbols: List[str]) -> Dict[str, PriceResult]:
    """
    Synchronous batch price fetch using Yahoo v7/finance/quote.
    Follows the exact pattern of _fetch_market_status_sync() (httpx.Client, _next_user_agent).
    Returns dict {symbol: PriceResult} for symbols with valid, non-zero price.
    Symbols absent from response or with invalid price are NOT in the return dict.
    Caller handles missing symbols (individual fallback path).
    """
    from api.integrations.market_data.market_data_settings import get_max_symbols_per_request
    batch_size = get_max_symbols_per_request()
    result: Dict[str, PriceResult] = {}

    for i in range(0, len(symbols), batch_size):
        batch = symbols[i:i + batch_size]
        symbols_param = ",".join(batch)
        url = "https://query1.finance.yahoo.com/v7/finance/quote"
        params = {
            "symbols": symbols_param,
            "fields": (
                "regularMarketPrice,regularMarketOpen,regularMarketDayHigh,"
                "regularMarketDayLow,regularMarketPreviousClose,"
                "regularMarketVolume,marketCap,regularMarketTime"
            ),
        }
        headers = {"User-Agent": _next_user_agent()}
        try:
            with httpx.Client(timeout=10) as client:
                resp = client.get(url, params=params, headers=headers)
                resp.raise_for_status()
                data = resp.json()
            quotes = (data or {}).get("quoteResponse", {}).get("result") or []
            for q in quotes:
                sym = q.get("symbol")
                raw_price = q.get("regularMarketPrice")
                if not sym or raw_price is None:
                    continue
                try:
                    price = Decimal(str(raw_price)).quantize(DECIMAL_SCALE)
                    if price <= 0:
                        continue
                    def _d(val):
                        return Decimal(str(val)).quantize(DECIMAL_SCALE) if val is not None else None
                    raw_time = q.get("regularMarketTime")
                    as_of = (
                        datetime.fromtimestamp(raw_time, tz=timezone.utc)
                        if raw_time else datetime.now(timezone.utc)
                    )
                    close_raw = q.get("regularMarketPreviousClose")
                    result[sym] = PriceResult(
                        symbol=sym,
                        price=price,
                        open_price=_d(q.get("regularMarketOpen")),
                        high_price=_d(q.get("regularMarketDayHigh")),
                        low_price=_d(q.get("regularMarketDayLow")),
                        close_price=_d(close_raw) if close_raw else price,
                        volume=int(q["regularMarketVolume"]) if q.get("regularMarketVolume") is not None else None,
                        market_cap=_d(q.get("marketCap")),
                        as_of=as_of,
                        provider="YAHOO_FINANCE",
                    )
                except (TypeError, ValueError):
                    continue
        except httpx.HTTPStatusError:
            raise  # caller handles 429/503 cooldown
        except Exception:
            continue  # network error on this batch; try remaining batches
    return result
```

**Step 2 — Add async wrapper inside `YahooProvider` class** (identical pattern to existing `get_ticker_price()`):

```python
async def get_ticker_prices_batch(self, symbols: List[str]) -> Dict[str, PriceResult]:
    """
    Async wrapper for _fetch_prices_batch_sync. Follows the exact pattern of
    get_ticker_price() — run_in_executor so sync httpx.Client runs in thread pool.
    REPLAY mode returns empty dict (no batch fixture); caller falls back to
    per-symbol replay path.
    """
    if not symbols:
        return {}
    if self._mode == "REPLAY":
        return {}  # batch has no replay fixture; caller falls back to per-symbol replay path
    loop = asyncio.get_event_loop()
    return await loop.run_in_executor(None, _fetch_prices_batch_sync, symbols)
```

```python
def is_in_cooldown(provider_name: str) -> bool:
    until = _cooldowns.get(provider_name)
    if not until:
        # For Alpha: check DB-persisted cooldown (survives restart)
        if provider_name == "ALPHA_VANTAGE":
            db_until = _read_alpha_cooldown_from_db()
            if db_until and time.time() < db_until:
                _cooldowns[provider_name] = db_until  # warm the in-memory cache
                return True
        return False
    if time.time() >= until:
        del _cooldowns[provider_name]
        return False
    return True

def _read_alpha_cooldown_from_db() -> Optional[float]:
    """Read alpha_cooldown_until from market_data.system_settings. Returns epoch float or None."""
    try:
        import psycopg2
        from api.integrations.market_data.market_data_settings import _get_sync_db_url
        url = _get_sync_db_url()
        if not url:
            return None
        conn = psycopg2.connect(url)
        try:
            cur = conn.cursor()
            cur.execute("SELECT value FROM market_data.system_settings WHERE key = 'alpha_cooldown_until'")
            row = cur.fetchone()
            if not row:
                return None
            from datetime import datetime, timezone
            dt = datetime.fromisoformat(row[0].replace("Z", "+00:00"))
            return dt.timestamp()
        finally:
            conn.close()
    except Exception:
        return None
```

## §10 ARCHITECTURAL CONSTRAINTS (IRON RULES — MUST NOT VIOLATE)

1. **NUMERIC(20,8) precision:** All price Decimals use `Decimal(str(val)).quantize(Decimal("0.00000001"))` — no float arithmetic, no rounding bypass.
2. **maskedLog compliance:** All new log statements must NOT include raw price values or API keys. Symbol names in DEBUG level only (not INFO or WARNING).
3. **Graceful degradation:** If Yahoo batch endpoint returns 429 → fallback to per-symbol individual fetch for that cycle, do not fail the job.
4. **No schema migrations:** WP003 adds zero new tables, columns, or migrations.
5. **REPLAY mode integrity:** `AlphaProvider(mode="REPLAY")` must not be affected by FIX-3. Replay mode returns fixtures without calling the provider API — `AlphaQuotaExhaustedException` must never be raised in REPLAY mode.
6. **No cooldown cascade on batch partial failure:** If Yahoo batch returns partial results (some symbols missing), tickers that got valid prices are committed. Only missing symbols enter the individual fallback path.
