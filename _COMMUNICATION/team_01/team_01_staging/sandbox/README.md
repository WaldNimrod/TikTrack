# 📦 Sandbox Directory

## מטרה
תיקייה זו מכילה את כל קבצי ה-**Preview/Sandbox** שנוצרו אוטומטית על ידי G-Bridge Validation System.

## מה יש כאן?
כל הקבצים שמתחילים ב-`_PREVIEW_` הם קבצי HTML שנוצרו אוטומטית לצורך:
- ✅ **ולידציה** - בדיקת עמידה בתקנים
- ✅ **תצוגה מקדימה** - בדיקת עיצוב לפני העברת לייצור
- ✅ **סנדבוקס** - סביבת בדיקה מבודדת

## קבצים מקוריים
הקבצים המקוריים נמצאים בתיקיית האב (`team_01_staging/`):
- `D15_INDEX.html` (לא `_PREVIEW_D15_INDEX.html`)
- `D16_ACCTS_VIEW.html` (לא `_PREVIEW_D16_ACCTS_VIEW.html`)
- `D18_BRKRS_VIEW.html` (לא `_PREVIEW_D18_BRKRS_VIEW.html`)
- וכו'...

## ⚠️ חשוב
- **אל תערוך** קבצים בתיקיית הסנדבוקס - הם נוצרים אוטומטית
- **ערוך רק** את הקבצים המקוריים בתיקיית האב
- קבצי ה-Preview מתעדכנים אוטומטית בעת הרצת G-Bridge Validation

## שימוש
פתח את קבצי ה-Preview בדפדפן כדי לבדוק את התוצאה לפני העברת לייצור.
