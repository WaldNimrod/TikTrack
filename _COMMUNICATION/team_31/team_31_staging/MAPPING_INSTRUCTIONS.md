# 🗺️ הוראות מיפוי טבלאות הלגסי - עמוד חשבונות מסחר

## 📋 מטרה
למפות את כל 5 הטבלאות בעמוד חשבונות מסחר מהלגסי כדי ליצור Blueprint מדויק.

## 🔧 כלים זמינים

### 1. סקריפט JavaScript לקונסולת דפדפן
**קובץ:** `scan-tables-browser-console.js`

**הוראות:**
1. פתח את הדפדפן וגש ל-`http://127.0.0.1:8090/trading_accounts` (התחבר עם admin/admin123)
2. לחץ F12 כדי לפתוח את הקונסולה
3. העתק והדבק את התוכן של `scan-tables-browser-console.js` לקונסולה
4. לחץ Enter
5. העתק את ה-JSON שמופיע בקונסולה
6. שמור את התוצאה בקובץ `trading-accounts-tables-mapping.json`

### 2. סקריפט Python (אם השרת רץ)
**קובץ:** `scan-trading-accounts-tables.py`

**הוראות:**
```bash
cd _COMMUNICATION/team_31/team_31_staging
python3 scan-trading-accounts-tables.py
```

התוצאה תישמר ב-`trading-accounts-tables-scan.json`

## 📊 מה צריך למפות

לכל אחת מ-5 הטבלאות, צריך למפות:

### Container 0: סיכום מידע והתראות פעילות
- [ ] שדות הטבלה (עמודות)
- [ ] סדר העמודות
- [ ] רינדור לכל שדה (פורמט, צבעים, יישור)
- [ ] פילטרים פנימיים (אם יש)
- [ ] וויגיטים נוספים (אם יש)

### Container 1: ניהול חשבונות מסחר (טבלה ראשונה)
- [ ] שדות הטבלה (עמודות) - **קריטי**
- [ ] סדר העמודות המדויק - **קריטי**
- [ ] רינדור מדויק לכל שדה:
  - שם חשבון - איך זה נראה?
  - ברוקר - איך זה נראה?
  - יתרה - פורמט, מטבע, יישור
  - מטבע - איך זה נראה?
  - סטטוס - badge? צבעים?
  - תאריך יצירה - פורמט תאריך
  - פעולות - אילו כפתורים/פעולות?
- [ ] חלוקה לעמודים (pagination) - איך זה נראה?
- [ ] מיון - איך זה נראה? (איקונים, מצבים)

### Container 2: סיכום תנועות לחשבון

### Container 3: דף חשבון לתאריכים

### Container 4: פוזיציות לפי חשבון
