# גיבוי קבצי CSS ישנים

## תיאור
תיקיית גיבוי זו מכילה קבצי CSS ישנים שהועברו מהמערכת הראשית.

## תאריך גיבוי

## קבצים כלולים

### styles/ (2 קבצים)
- js-map.css - סגנונות למפת JavaScript
- page-scripts-matrix.css - סגנונות למטריצת סקריפטים
- entity-details-system/ - תיקיית משנה לסגנונות מערכת פרטי ישויות

### dist/ (1 קובץ)
- main.css - קובץ CSS ראשי מותאם (159KB)

### remaining-styles.css
- סגנונות שלא הועברו למערכת ITCSS החדשה
- כולל הגדרות טבלאות, מפרידי כפתורים, וסגנונות נוספים

## מערכת נוכחית
המערכת הנוכחית משתמשת ב-ITCSS architecture עם 51 קבצי CSS מאורגנים ב-9 שכבות.

## הערות
- קבצים אלו אינם בשימוש במערכת הנוכחית
- ניתן למחוק את התיקייה לאחר וידוא שהמערכת החדשה עובדת תקין
- לגישה לקבצים, יש להעתיק אותם בחזרה לתיקיית trading-ui
