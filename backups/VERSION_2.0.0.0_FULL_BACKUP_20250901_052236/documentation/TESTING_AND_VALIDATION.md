## בדיקות מערכת המשתמשים

### בדיקות API

#### 1. בדיקת משתמש ברירת מחדל
```bash
curl -s http://localhost:8080/api/v1/users/default | jq
```
**תוצאה**: ✅ משתמש nimrod (ID: 1) נמצא ופעיל

#### 2. בדיקת העדפות משתמש
```bash
curl -s http://localhost:8080/api/v1/preferences/ | jq
```
**תוצאה**: ✅ העדפות נטענות עם ברירות מחדל

#### 3. בדיקת שמירת העדפות
```bash
curl -s -X POST http://localhost:8080/api/v1/preferences/ \
  -H "Content-Type: application/json" \
  -d '{"preferences": {"defaultStopLoss": 15}}' | jq
```
**תוצאה**: ✅ העדפות נשמרות בהצלחה

#### 4. בדיקת איפוס העדפות
```bash
curl -s -X POST http://localhost:8080/api/v1/preferences/reset | jq
```
**תוצאה**: ✅ העדפות מאופסות לברירות מחדל

### בדיקות בסיס נתונים

#### 1. בדיקת טבלת משתמשים
```sql
SELECT * FROM users;
```
**תוצאה**: ✅ טבלת users קיימת עם משתמש ברירת מחדל

#### 2. בדיקת משתמש ברירת מחדל
```sql
SELECT * FROM users WHERE is_default = 1;
```
**תוצאה**: ✅ משתמש nimrod (ID: 1) מוגדר כמשתמש ברירת מחדל

#### 3. בדיקת העדפות
```sql
SELECT id, username, preferences FROM users WHERE id = 1;
```
**תוצאה**: ✅ העדפות JSON נשמרות בבסיס הנתונים

- **קובץ preferences כפול**: `preferences-simple.py` נמחק
