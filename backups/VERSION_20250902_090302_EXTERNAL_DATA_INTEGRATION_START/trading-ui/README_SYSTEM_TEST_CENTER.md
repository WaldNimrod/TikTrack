# מרכז בדיקות מערכת - TikTrack

## סקירה כללית

מרכז בדיקות המערכת הוא דף מאוחד המספק ממשק אחד לבדיקת כל המערכות והשירותים של TikTrack. הדף מחולק לסקשנים שונים עם ניווט נוח בין המערכות השונות.

## כתובת גישה

```
/system-test-center
```

**הערה**: הדף נגיש דרך התפריט הראשי → הגדרות → נתונים חיצוניים

### ניווט בדף:
- **תפריט ראשי מלא** עם כל העמודים של TikTrack
- **ניווט מהיר** בין כל המערכות
- **עיצוב עקבי** עם שאר האתר

## מבנה הדף

### 1. סקירת סטטוס מערכות (System Status Overview)
- **מערכת Cache**: מציג סטטוס בריאות, שימוש זיכרון
- **Query Optimization**: מציג ציון ביצועים ומספר queries
- **נתונים חיצוניים**: מציג סטטוס API וספקים
- **ביצועים**: מציג זמני תגובה כללים

### 2. סקשן מערכת Cache
- **סטטיסטיקות Cache**: רשומות, אחוז פגיעות, שימוש זיכרון
- **בדיקת בריאות**: סטטוס מערכת, בדיקות זיכרון ו-hit rate
- **פעולות Cache**: ניקוי, invalidation לפי dependencies
- **מידע על המערכת**: תכונות, גרסה, סטטוס

### 3. סקשן Query Optimization
- **סקירת ביצועים**: סטטיסטיקות queries, זמני ביצוע
- **הזדמנויות אופטימיזציה**: queries עם פוטנציאל שיפור
- **Queries איטיים**: רשימת queries איטיים עם הצעות
- **פעולות ניהול**: ניקוי פרופילים, ייצוא נתונים
- **בדיקת Queries**: הרצת בדיקות על סוגי queries שונים

### 4. סקשן נתונים חיצוניים
- **סטטוס API**: בדיקת זמינות מערכת הנתונים החיצוניים
- **בדיקת מודלים**: בדיקת מודלי נתונים וספקים
- **בדיקת נתונים**: קבלת ציטוטים בודדים ומרובים
- **בדיקת ספקים**: בדיקת זמינות ספקי נתונים

### 5. סקשן ביצועים
- **בדיקת ביצועים כללית**: הרצת בדיקות עם מספר איטרציות
- **מדדי מערכת**: מדדים כלליים של ביצועי המערכת
- **בדיקת זמני תגובה**: בדיקת זמני תגובה של APIs שונים

### 6. יומן פעילות מאוחד
- **יומן מאוחד**: כל הפעולות מכל המערכות במקום אחד
- **ניהול יומן**: ניקוי וייצוא יומן הפעילות
- **סינון לפי רמה**: info, success, warning, error

## שימוש

### כניסה לדף
1. גש לכתובת `/system-test-center`
2. הדף יטען אוטומטית את סטטוס כל המערכות
3. השתמש בטאבים לניווט בין המערכות השונות

### בדיקת מערכת
1. בחר את המערכת הרצויה מהטאבים
2. השתמש בכפתורי "רענן" לעדכון נתונים
3. בצע פעולות ניהול לפי הצורך
4. עקוב אחר היומן לפעילות המערכת

### ניהול יומן
1. השתמש בכפתור "נקה יומן" לניקוי
2. השתמש בכפתור "ייצא יומן" לייצוא
3. היומן מתעדכן אוטומטית עם כל פעולה
