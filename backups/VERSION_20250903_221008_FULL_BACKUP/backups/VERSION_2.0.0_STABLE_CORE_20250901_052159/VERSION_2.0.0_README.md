# TikTrack גרסה 2.0.0 - הגרסה היציבה הראשונה

## מידע על הגרסה
- **מספר גרסה**: 2.0.0
- **תאריך יצירה**: 1 בספטמבר 2025
- **סוג**: גרסה יציבה ראשונה של ליבת המערכת
- **תיאור**: גרסה יציבה ומנוקה מקבצים זמניים

## שינויים עיקריים
- ניקוי קבצים זמניים וקבצי בדיקות
- שמירת קבצים חשובים ומקושרים מהתפריט
- עדכון התפריט הראשי עם קישור "מיפוי סקריפטים"
- שחזור קבצים חשובים מגיט האב

## קבצים שנשמרו
- page-scripts-matrix.html
- trading-ui/js-map.html
- trading-ui/test-header-only.html
- trading-ui/tests.html
- Backend/create_fresh_database.py
- כל תיקיות external_data_integration_client

## קבצים שנמחקו
- קבצי ניקיון ותיעוד זמניים
- קבצי בדיקות וסקריפטים זמניים
- קבצי לוגים ובקאפים ישנים
- תיקיות ריקות

## הוראות הפעלה
1. הפעל את השרת: `./start_dev.sh`
2. פתח בדפדפן: `http://localhost:8080`
3. גש לתפריט "בדיקות" -> "מיפוי סקריפטים" לבדיקת מיפוי

## הערות
- זו הגרסה היציבה הראשונה של ליבת המערכת
- כל הקבצים המקושרים מהתפריט הראשי נשמרו
- המערכת מוכנה לשימוש יצרני
