## שינויים עיקריים בגרסה זו

### 1. תיקון מודול התראות
- **תיקון מיפוי שדות בטבלה**: החלפת סדר עמודות "סוג התראה" ו"הודעה"
- **תיקון כפתורי פעולה**: שינוי מ-column ל-row
- **תיקון הודעות**: שימוש במערכת ההודעות הרגילה במקום מודלים חדשים
- **הוספת צבע רקע למודל**: רקע אדום לכותרת מודל התראות

### 2. הסרת הגבלות על התראות
- **סוגי התראות**: כל הסוגים נתמכים (מחיר, סטופ לוס, נפח, מותאם אישית)
- **תכונות תנאי**: כל התכונות נתמכות (מחיר, שינוי, ממוצע נע, נפח)
- **אופרטורים**: כל האופרטורים נתמכים ללא הגבלה
- **הסרת הודעות "בפיתוח"**: אין יותר הודעות מטרידות

### 3. תיקון z-index למודלים
- **הגדרת z-index גבוה**: וידוא שמודלים מופיעים מעל כל התוכן
- **תיקון שכבות**: מודלים, הודעות, וטוסטים מופיעים בסדר הנכון

### 4. תיקון מיפוי עמודות
- **עדכון table-mappings.js**: התאמה למבנה הטבלה האמיתי
- **תיקון פונקציית getColumnValue**: טיפול נכון בשדות מורכבים
- **תיקון תרגום תנאים**: שימוש בפונקציית translateConditionFields

### 5. הוספת event listeners
- **שדות תנאי**: event listeners למודל הוספה ועריכה
- **בדיקת תקינות**: וולידציה בזמן אמת
- **הודעות מותאמות**: הודעות ברורות למשתמש
