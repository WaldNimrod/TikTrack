📊 TikTrack Project Snapshot

מסמך מסכם מקיף של מערכת ניהול המסחר
תאריך: 15 בינואר 2025 | גרסה: 2.0.0

🎯 סקירה כללית

מטרת המערכת

מערכת ניהול מסחר מקיפה לניהול תיקי השקעות, מעקב אחר טריידים, תכנון עסקאות וניתוח ביצועים.

טכנולוגיות

  • Backend: Python Flask + SQLAlchemy
  • Frontend: HTML5 + CSS3 + JavaScript
  • Database: SQLite עם WAL mode
  • UI Framework: AG Grid + Bootstrap

סטטוס פיתוח

  • Backend API מוכן
  • Frontend UI מוכן
  • Authentication בפיתוח
  • Testing נדרש

users משתמשי המערכת username, email, password_hash, is_active מוכן
roles תפקידים והרשאות name, description, permissions מוכן

tickers מניות וניירות ערך symbol, name, type, currency, active_trades מוכן
accounts חשבונות מסחר name, currency, status, cash_balance, total_value מוכן
trades עסקאות מסחר account_id, ticker_id, status, type, total_pl מוכן
trade_plans תכנוני עסקאות account_id, ticker_id, investment_type, planned_amount מוכן

alerts התראות מערכת type, condition, message, is_active מוכן
cash_flows תזרימי מזומנים account_id, type, amount, date מוכן
notes הערות ומסמכים content, attachment, related_entity מוכן

אינדקסים ויחסים

אינדקסים אסטרטגיים: נוצרו אינדקסים על שדות שנשאלים לעתים קרובות
  • tickers: symbol (unique), active_trades
  • trades: account_id, ticker_id, status, opened_at
  • accounts: status, currency
  • users: username, email (unique)

Backend Services

TickerService

  • get_all_tickers() - קבלת כל המניות
  • get_ticker_by_id(id) - קבלת מניה לפי ID
  • create_ticker(data) - יצירת מניה חדשה
  • update_ticker(id, data) - עדכון מניה
  • delete_ticker(id) - מחיקת מניה

AccountService

  • get_all_accounts() - קבלת כל החשבונות
  • get_account_by_id(id) - קבלת חשבון לפי ID
  • create_account(data) - יצירת חשבון חדש
  • update_account(id, data) - עדכון חשבון
  • get_account_stats(id) - סטטיסטיקות חשבון

TradeService

  • get_all_trades() - קבלת כל העסקאות
  • get_trades_by_account(account_id) - עסקאות לפי חשבון
  • create_trade(data) - יצירת עסקה חדשה
  • update_trade(id, data) - עדכון עסקה
  • close_trade(id) - סגירת עסקה

TradePlanService

  • get_all_plans() - קבלת כל התכנונים
  • get_plans_by_account(account_id) - תכנונים לפי חשבון
  • create_plan(data) - יצירת תכנון חדש
  • update_plan(id, data) - עדכון תכנון
  • cancel_plan(id, reason) - ביטול תכנון

Frontend JavaScript Objects

Grid System

  • initializeGridSystem() - אתחול מערכת גריד
  • createGrid(containerId, data) - יצירת גריד חדש
  • updateGridData(newData) - עדכון נתוני גריד
  • applyFilters(filters) - החלת פילטרים
  • exportGridData(format) - ייצוא נתונים

Authentication

  • login(username, password) - התחברות
  • logout() - התנתקות
  • refreshToken() - רענון טוקן
  • checkAuth() - בדיקת אימות
  • getCurrentUser() - קבלת משתמש נוכחי

API Communication

  • apiCall(endpoint, method, data) - קריאת API
  • handleApiError(error) - טיפול בשגיאות
  • retryApiCall(call, maxRetries) - ניסיון חוזר
  • showNotification(message, type) - הצגת הודעות

Data Management

  • loadTableData(tableName) - טעינת נתוני טבלה
  • saveTableData(tableName, data) - שמירת נתונים
  • validateForm(formData) - אימות טופס
  • formatCurrency(amount) - עיצוב מטבע
  • formatDate(date) - עיצוב תאריך
