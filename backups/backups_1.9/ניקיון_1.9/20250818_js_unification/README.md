# גיבוי איחוד קבצי JavaScript - 18 באוגוסט 2025

## 📋 תיאור
גיבוי זה מכיל את הקבצים הישנים לפני איחוד קבצי JavaScript בגרסה 2.3.

## 📁 קבצים בגיבוי

### 🔄 קבצים שאוחדו
- **`grid-table.js.backup`** - הקובץ הישן של פונקציות הגריד הכלליות
- **`grid-data.js.backup`** - הקובץ הישן של ניהול נתונים
- **`grid-init.js.backup`** - הקובץ הישן של אתחול הגריד
- **`main.js.backup`** - הקובץ המקורי של main.js לפני האיחוד

### 🎯 איחוד שבוצע
כל הפונקציונליות מ-`grid-table.js`, `grid-data.js` ו-`grid-init.js` אוחדה לתוך `main.js` החדש.

## 📝 שינויים שבוצעו

### ✅ מה השתנה
1. **איחוד קבצים:** `grid-table.js` + `grid-data.js` + `grid-init.js` → `main.js`
2. **עדכון דפים:** כל הדפים מעודכנים לטעון `main.js`
3. **הגדרת כללים:** פונקציות כלליות ב-`main.js`, פונקציות ספציפיות בקבצים ייעודיים

### 🔧 כללי פיתוח חדשים
1. **פונקציות כלליות** → `main.js`
2. **פונקציות ספציפיות לישות** → קובץ ייעודי (למשל `accounts.js`)
3. **אין צורך** לטעון את הקבצים הישנים
