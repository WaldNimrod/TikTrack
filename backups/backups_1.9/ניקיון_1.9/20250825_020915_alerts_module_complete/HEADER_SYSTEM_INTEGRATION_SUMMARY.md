## 🎯 מטרה

## 🔧 שינויים טכניים עיקריים

### 1. החלפת Header
```html
<!-- לפני -->
<app-header></app-header>
<script src="scripts/app-header.js"></script>

<!-- אחרי -->
<!-- כותרת האפליקציה החדשה - נוצרת אוטומטית על ידי header-system.js -->
<!-- unified-header ייווצר כאן אוטומטית -->
```

### 2. הוספת CSS ו-JavaScript
```html
<!-- CSS -->
<link rel="stylesheet" href="styles/header-system.css">

<!-- JavaScript -->
<script src="scripts/translation-utils.js"></script>
<script src="scripts/header-system.js"></script>
```

### 3. עדכון אתחול
```javascript
// לפני
if (typeof window.HeaderSystem !== 'undefined') {
  window.headerSystem = new window.HeaderSystem();
  window.headerSystem.initialize();
}

// אחרי
if (window.headerSystem && !window.headerSystem.isInitialized) {
  console.log('✅ אתחול HeaderSystem...');
  window.headerSystem.init();
}
```

### 4. הוספת Data Attributes לטבלאות

## 🎨 תכונות הפילטר החדש

### פילטרים זמינים
1. **פילטר סטטוס** - בחירה מרובה (פתוח, סגור, מבוטל, פעיל, לא פעיל, הושלם, ממתין)
2. **פילטר טיפוס** - בחירה מרובה (סווינג, השקעה, פסיבי, מניה, ETF, אג"ח, קריפטו, קנייה, מכירה, הפקדה, משיכה, דיבידנד, עמלה, ריבית)
3. **פילטר חשבון** - בחירה מרובה (דינמי מהשרת)
4. **פילטר תאריכים** - בחירה יחידה (היום, אתמול, השבוע, וכו')
5. **פילטר חיפוש** - חיפוש טקסט בזמן אמת

### כפתורי פעולה
- **כפתור איפוס** (↻) - מנקה את כל הפילטרים
- **כפתור ניקוי** (×) - סוגר את כל התפריטים
- **כפתור הצג/הסתר** (▼/▶) - מציג/מסתיר את אזור הפילטרים

### שמירת מצב
- שמירת מצב הפילטרים ב-Local Storage
- שמירת מצב פתוח/סגור של אזור הפילטרים
- שמירת מצב הניווט הפעיל
