## 🚀 **הרצת בדיקות מהירה:**

### 📋 **פקודות Makefile:**
```bash
cd Backend/testing_suite

# עזרה
make help

# התקנת תלויות
make install

# הרצת כל הבדיקות
make test

# בדיקות בטוחות (מומלץ)
make test-safe

# בדיקות עם גיבוי
make test-with-backup

# בדיקות ספציפיות
make test-unit          # בדיקות יחידה
make test-integration   # בדיקות אינטגרציה
make test-e2e          # בדיקות end-to-end
make test-performance  # בדיקות ביצועים
make test-load         # בדיקות עומס
make test-security     # בדיקות אבטחה
```

### 🔧 **פקודות pytest ישירות:**
```bash
cd Backend/testing_suite

# כל הבדיקות
python3 -m pytest -v

# בדיקות בטוחות
python3 -m pytest -v --tb=short --strict-markers --disable-warnings

# בדיקות עם כיסוי
python3 -m pytest -v --cov=.. --cov-report=html

# בדיקות ספציפיות
python3 -m pytest unit_tests/ -v
python3 -m pytest integration_tests/ -v
python3 -m pytest e2e_tests/ -v
```
