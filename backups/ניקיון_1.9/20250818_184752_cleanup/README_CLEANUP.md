## מטרת הניקוי
ניקוי המערכת מקבצים מיותרים, כפולים וזמניים כדי לשפר את הביצועים והארגון.

## קבצים שהועברו לגיבוי

### קבצי סקריפטים מיותרים:
- **add_*.py** - סקריפטים להוספת נתונים (כבר לא נדרשים)
- **fix_*.py** - סקריפטים לתיקון נתונים (כבר לא נדרשים)
- **update_*.py** - סקריפטים לעדכון נתונים (כבר לא נדרשים)
- **remove_*.py** - סקריפטים להסרת נתונים (כבר לא נדרשים)
- **test_*.py** - קבצי בדיקה (לא נדרשים בייצור)

### קבצי SQL מיותרים:
- **add_*.sql** - סקריפטי SQL להוספת נתונים
- **fix_*.sql** - סקריפטי SQL לתיקון נתונים
- **update_*.sql** - סקריפטי SQL לעדכון נתונים
- **seed_*.sql** - סקריפטי SQL לזריעת נתונים

### תיקיות מיותרות:
- **tests/** - תיקיית בדיקות (לא נדרשת בייצור)
- **migrations/** - תיקיית מיגרציות (כבר לא נדרשת)

### קבצי גיבוי בסיס נתונים:
- **simpleTrade_new_backup_*.db** - גיבויים ישנים של בסיס הנתונים
- **simpleTrade_new_backup_*.db-shm** - קבצי WAL ישנים
- **simpleTrade_new_backup_*.db-wal** - קבצי WAL ישנים
