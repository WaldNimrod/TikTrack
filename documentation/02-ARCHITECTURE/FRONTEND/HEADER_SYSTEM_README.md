# Header System - README

## Overview

מערכת Header מאוחדת עם תפריט ראשי ופילטרים.

## Files

- `trading-ui/scripts/header-system.js` - ליבה של המערכת

## Features

- תפריט ראשי מאוחד
- פילטרים גלובליים
- שמירת מצב
- תמיכה מלאה ב-RTL

## Architecture

המערכת בנויה כרכיב Web Component עצמאי:

- `<unified-header>` - רכיב ראשי
- אינטגרציה עם Filter System
- אינטגרציה עם Menu System

## Status

✅ **ACTIVE** - Used on all main pages
