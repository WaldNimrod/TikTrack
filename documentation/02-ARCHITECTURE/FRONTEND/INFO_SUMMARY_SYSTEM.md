# Info Summary System

## Overview

חישובי KPI ותמיכה בסינונים עם רינדור RTL.

## Files

- `trading-ui/scripts/info-summary-system.js`
- `trading-ui/scripts/services/statistics-calculator.js`

## Features

- חישובי KPI אוטומטיים
- תמיכה מלאה בסינונים
- רינדור RTL תואם
- סטטיסטיקות בזמן אמת

## KPIs Available

- סכום עסקאות
- רווח/הפסד
- ביצועים לפי תקופה
- סטטיסטיקות לפי סוג נכס
- ועוד
