## ⚠️ דרישות אטריביוציה

### חובה להוסיף

לפי הרישיון, **חובה** להוסיף:

1. **ההודעה מ-NOTICE**:
   ```
   TradingView Lightweight Charts
   Copyright (c) TradingView, Inc.
   Licensed under Apache License 2.0
   ```

2. **קישור ל-TradingView**:
   - קישור ל-https://www.tradingview.com/
   - או קישור ל-https://www.tradingview.com/lightweight-charts/

### מיקום האטריביוציה

האטריביוציה חייבת להופיע ב:
- ✅ **דף הציבורי** של האתר או האפליקציה
- ✅ **עמוד About** או **Credits**
- ✅ **Footer** של האתר
- ✅ **README** של הפרויקט

### דוגמת אטריביוציה

```html
<!-- Footer -->
<footer>
    <p>
        Charts powered by 
        <a href="https://www.tradingview.com/lightweight-charts/" target="_blank" rel="noopener">
            TradingView Lightweight Charts
        </a>
    </p>
</footer>
```

או:

```markdown
## Credits

- **Charts**: [TradingView Lightweight Charts](https://www.tradingview.com/lightweight-charts/)
  - Copyright (c) TradingView, Inc.
  - Licensed under Apache License 2.0
```
