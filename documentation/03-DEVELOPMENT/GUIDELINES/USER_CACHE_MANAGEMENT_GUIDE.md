## 📋 מתי בכל זאת צריך לנקות מטמון

**דוגמה:** שינית צבעי מערכת או העדפות תצוגה

**פתרון:**

```

```

זהו! המטמון ינוקה והנתונים יתעדכנו מיד.

**❌ אל תעשה:** F5 / Cmd+R / hard refresh  
**✅ עשה:** לחץ 🧹 (פשוט ומהיר)

---

### 2️⃣ **אם רואה משהו מוזר** (נדיר מאוד)

**דוגמה:**

- טבלה לא מתעדכנת אחרי עריכה
- מספרים לא מדויקים
- כפתורים לא מגיבים

**פתרון:**

```
1. לחץ 🧹 בתפריט (ניקוי בינוני)

```

**אם עדיין לא עזר:**

```

```

---

### 3️⃣ **במקרי חירום בלבד** (כמעט אף פעם)

**דוגמה:**

- המערכת תקועה לגמרי
- שום דבר לא עובד
- צריך reset מוחלט

**פתרון:**

```
1. פתח: כלי פיתוח → ניהול מערכת
2. לחץ: ☢️ Nuclear - ניקוי גרעיני
3. קרא את האזהרה!
4. אשר
5. המערכת תתרענן אוטומטית
6. תצטרך להתחבר מחדש (אם יש אימות)
```

⚠️ **אזהרה:** זה מוחק **הכל** - כולל העדפות אישיות!
