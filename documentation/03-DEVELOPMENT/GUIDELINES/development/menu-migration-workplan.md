# תוכנית עבודה שיטתית - מיגרציית תפריט TikTrack

## מטרה

העברת תפריט TikTrack מהמערכת הישנה למערכת החדשה עם Bootstrap 5, תוך שמירה על כל הפונקציונליות.

## עקרונות עבודה

1. **עבודה שלבית** - כל שלב חייב להסתיים לפני המעבר לבא
2. **בדיקות ביניים** - כל שלב נבדק לפני המשך
3. **גיבוי אוטומטי** - כל שלב נשמר בגיט
4. **אין חזרות** - כל שלב מבוצע פעם אחת בלבד
5. **תיעוד מלא** - כל שינוי מתועד

## תהליך גיבוי

### כל שלב

1. `git add .`
2. `git commit -m "שלב X: תיאור השלב"`
3. `git push`

### במקרה של בעיה

1. `git log --oneline` - לראות את ההיסטוריה
2. `git reset --hard HEAD~1` - לחזור לשלב הקודם
3. `git push --force` - לעדכן את המאגר
