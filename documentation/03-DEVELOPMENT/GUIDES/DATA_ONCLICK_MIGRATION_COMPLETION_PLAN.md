#### צעד 4.2: המרה של כפתורים סטטיים

**דוגמה - לפני:**

```html
<button onclick="editRecord(123)">ערוך</button>
```

**דוגמה - אחרי:**

```html
<button data-button-type="EDIT" 
        data-onclick="editRecord(123)" 
        data-text="ערוך">
</button>
```

#### צעד 4.3: המרה של כפתורים דינמיים

**דוגמה - לפני:**

```javascript
const button = document.createElement('button');
button.onclick = () => editRecord(123);
button.textContent = 'ערוך';
```

**דוגמה - אחרי:**

```javascript
const button = document.createElement('button');
button.setAttribute('data-button-type', 'EDIT');
button.setAttribute('data-onclick', 'editRecord(123)');
button.setAttribute('data-text', 'ערוך');
// או שימוש ב-Button System:
const buttonHtml = window.ButtonSystem.createButton({
    type: 'EDIT',
    onclick: 'editRecord(123)',
    text: 'ערוך'
});
```
