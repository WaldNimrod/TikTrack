## ✅ מערכת פוזיציות לפי חשבון

### 3. בחירת חשבון
- [ ] רשימה נפתחת עם כל החשבונות
- [ ] חשבון ברירת מחדל נבחר אוטומטית
- [ ] בחירת חשבון אחר מעדכנת את הטבלה
- [ ] הטבלה מציגה פוזיציות של החשבון הנבחר

### 4. כותרת "פוזיציות לפי חשבון"
- [ ] מספר הפוזיציות מוצג נכון
- [ ] מופיע מפריד `|` אחרי המספר
- [ ] "סה״כ שווי פוזיציות" מוצג (אם יש פוזיציות)
- [ ] "סה״כ שווי חשבון" מוצג (אם יש פוזיציות)
- [ ] הערכים מפורמטים נכון (פסיקים, סימן בסוף)

### 5. טבלת פוזיציות
- [ ] העמודה הראשונה: "סימבול" (קישור לפרטי טיקר)
- [ ] העמודה השנייה: "נוכחי" (מידע עדכני של הטיקר)
- [ ] כל העמודות מוצגות נכון:
  - [ ] כמות (עם סימן #, חיובי/שלילי)
  - [ ] מחיר ממוצע
  - [ ] שווי שוק (ללא ספרות אחרי נקודה)
  - [ ] רווח/הפסד לא מוכר (ללא ספרות אחרי נקודה)
  - [ ] אחוז מחשבון
  - [ ] אחוז מהתיק
- [ ] כל הערכים הכספיים מפורמטים נכון (פסיקים, סימן בסוף)
- [ ] אחוזים עם סימן +/- בסוף (RTL)

### 6. סידור בטבלת פוזיציות
- [ ] לחיצה על "סימבול" מסדרת את הטבלה
- [ ] לחיצה על "נוכחי" מסדרת את הטבלה
- [ ] לחיצה על "כמות" מסדרת את הטבלה
- [ ] לחיצה על כל עמודה אחרת מסדרת את הטבלה
- [ ] הטבלה מתעדכנת מיד אחרי הסידור
- [ ] אין שגיאות בקונסולה בעת סידור

### 7. מודול פרטי פוזיציה
- [ ] לחיצה על שורה בטבלה פותחת מודול פרטים
- [ ] שורה ראשונה: סימבול | צד | חשבון
- [ ] סימבול הוא קישור לפרטי טיקר
- [ ] עמודה ראשונה: כמות, מחיר ממוצע, סהכ עלות, שווי שוק, אחוז מחשבון
- [ ] עמודה שנייה: P/L לא מוכר, רווח/הפסד מוכר, סה״כ P/L, סה״כ קניות, סה״כ מכירות, עמלות
- [ ] כמות מוצגת עם # (חיובי/שלילי)
- [ ] מחיר ממוצע מציג ברוטו ונטו בסוגריים
- [ ] כל הערכים הכספיים מפורמטים נכון
- [ ] טבלת ביצועים מוצגת במודול
- [ ] סידור בטבלת ביצועים עובד
- [ ] כפתור סגירה (איקון) בסוף שורת הכותרת משמאל
- [ ] צבעי המודול הם "trade entity colors"
