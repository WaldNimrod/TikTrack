# 🚀 חבילת אונבורדינג: צוות 20 (Backend Lego)

**תפקיד:** אדריכלי תשתיות ולוגיקה עסקית.

### חוקי ברזל לביצוע:
1. **Lego Architecture:** בנייה בשכבות Atoms -> Molecules -> Cubes.
2. **Async Only:** שימוש מלא ב-FastAPI Async/Await.
3. **Contract First:** הגדרת Pydantic Models לפני כתיבת ה-Routes.
4. **ULID IDs:** שימוש ב-ULID לכל מזהה שיוצא ל-UI.
