## 🎯 סקירה כללית

מערכת AI Analysis מאפשרת לך ליצור ניתוחים מפורטים של מניות באמצעות בינה מלאכותית. המערכת תומכת ב-4 תבניות ניתוח שונות ומאפשרת שימוש במנועי AI שונים (Gemini או Perplexity).

### תכונות עיקריות

- **4 תבניות ניתוח** - ניתוח מחקר הון, ניתוח טכני, ניתוח ביצועים, ניתוח סיכונים
- **מנועי AI מרובים** - תמיכה ב-Gemini ו-Perplexity
- **API Keys אישיים** - כל משתמש מגדיר את המפתחות שלו
- **היסטוריה מלאה** - שמירת כל הניתוחים להיסטוריה
- **אינפוגרפיקות** - הצגת תוצאות עם גרפים וטבלאות

---

## 🚀 התחלה מהירה

### שלב 1: הגדרת API Key

1. עבור לעמוד **פרופיל משתמש**
2. גלול לסקשן **"הגדרות AI Analysis"**
3. הזן את ה-API key שלך (Gemini או Perplexity)
4. לחץ על **"שמור"**
5. המערכת תבדוק את המפתח אוטומטית

### שלב 2: יצירת ניתוח ראשון

1. עבור לעמוד **AI Analysis**
2. לחץ על **"צור ניתוח חדש"** בכותרת העמוד
3. **שלב 1:** בחר מנוע AI, שפת משוב ותבנית ניתוח
4. **שלב 2:** מלא את הפילטרים (אם יש) ואת המשתנים הנדרשים
5. **שלב 3:** צפה בתוצאות הניתוח
6. תוכל לשמור כהערה או לייצא את התוצאות

> **הערה:** המערכת משתמשת בממשק וויזארד חדש עם 3 שלבים, בדומה לתהליך ייבוא נתוני משתמש.

## 📝 תבניות הפרומפטים

### תבנית 1: ניתוח מחקר הון (Equity Research Analysis)

**מתי להשתמש:**
ניתוח מפורט של מניה עם פרספקטיבה פונדמנטלית ומקרו-כלכלית.

**משתנים:**

- **Stock Ticker / Company Name** - שם המניה או החברה
- **Investment Thesis** - תזה השקעה
- **Goal** - מטרת הניתוח

**תוצאות:**

- ניתוח פונדמנטלי (Revenue, Margins, Cash Flow)
- השוואת הערכה מול מתחרים
- סקירת בעלות פנימית
- אימות תזה (3 טיעונים תומכים, 2 נגד)
- סקירת סקטור ומקרו
- מיקום תחרותי
- רשימת קטליזטורים
- סיכום השקעה והמלצה

---

### תבנית 2: ניתוח טכני מעמיק (Technical Analysis Deep Dive)

**מתי להשתמש:**
ניתוח טכני מפורט עם אינדיקטורים ודפוסי גרף.

**משתנים:**

- **Stock Ticker / Company Name** - שם המניה
- **Time Frame** - טווח זמן (1 יום / 1 שבוע / 1 חודש / 3 חודשים / 1 שנה)
- **Technical Indicators** - בחירה מרשימת אינדיקטורים טכניים
- **Chart Pattern Focus** - דפוסי גרף (Head & Shoulders, Triangles, Flags, וכו')
- **Investment Type** - סוג השקעה (סווינג, השקעה, פאסיבי, וכו')

**תוצאות:**

- ניתוח תנועת מחיר
- סקירת אינדיקטורים טכניים (RSI, MACD, Moving Averages)
- זיהוי דפוסי גרף
- ניתוח תמיכה והתנגדות
- הערכת setup מסחר (Entry, Stop-Loss, Take-Profit)
- סיכום טכני והמלצה

---

### תבנית 3: ניתוח ביצועים ופורטפוליו (Trade Performance & Portfolio Analysis)

**מתי להשתמש:**
ניתוח ביצועי המסחר שלך וזיהוי דפוסים.

**משתנים:**

- **Ticker Symbol** - טיקר לניתוח
- **Date Range** - טווח תאריכים
- **Analysis Focus** - מוקד ניתוח (Performance Review / Risk Assessment / Optimization)
- **Investment Type Filter** - סינון לפי סוג השקעה
- **Trading Account** - סינון לפי חשבון מסחר (אופציונלי)

**תוצאות:**

- סקירת ביצועים (P&L, Win Rate, ROI)
- ניתוח דפוסי מסחר
- הערכת סיכונים
- סקירת איכות ביצוע
- המלצות לאופטימיזציה
- תוכנית פעולה

---

### תבנית 4: ניתוח סיכונים ותנאים (Risk & Conditions Analysis)

**מתי להשתמש:**
ניתוח יעילות תנאי המסחר וניהול סיכונים.

**משתנים:**

- **Ticker Symbol** - טיקר לניתוח
- **Trade Plan ID** - ניתוח תנאי תוכנית (אופציונלי)
- **Trade ID** - ניתוח תנאי טרייד (אופציונלי)
- **Condition Focus** - בחירה משיטות המסחר
- **Investment Type** - סוג השקעה

**תוצאות:**

- ניתוח יעילות תנאים
- הערכת פרופיל סיכון
- המלצות לאופטימיזציה
- תוכנית פעולה

## 🔑 ניהול API Keys

### Gemini API Key

**איך להשיג:**

1. עבור ל-[Google AI Studio](https://aistudio.google.com/)
2. התחבר עם חשבון Google שלך
3. לחץ על **"Get API Key"**
4. בחר **"Create API Key"**
5. העתק את המפתח

**הזנה במערכת:**

1. עבור לפרופיל משתמש
2. גלול לסקשן **"הגדרות AI Analysis"**
3. הדבק את המפתח בשדה **"Gemini API Key"**
4. לחץ על **"שמור"**
5. המערכת תבדוק את המפתח אוטומטית

### Perplexity API Key

**איך להשיג:**

1. עבור ל-[Perplexity API](https://www.perplexity.ai/api)
2. התחבר או צור חשבון
3. עבור ל-**"API Keys"**
4. לחץ על **"Create API Key"**
5. העתק את המפתח

**הזנה במערכת:**

1. עבור לפרופיל משתמש
2. גלול לסקשן **"הגדרות AI Analysis"**
3. הדבק את המפתח בשדה **"Perplexity API Key"**
4. לחץ על **"שמור"**
5. המערכת תבדוק את המפתח אוטומטית

### בחירת מנוע ברירת מחדל

בסקשן הגדרות AI Analysis, תוכל לבחור איזה מנוע AI ישמש כברירת מחדל:

- **Gemini** - מומלץ למתחילים (חינמי, יציב)
- **Perplexity** - מתאים לניתוחים עם חיפוש אינטרנט

## 📊 יצירת ניתוח

המערכת משתמשת בממשק וויזארד עם 3 שלבים לניהול תהליך יצירת הניתוח.

### פתיחת הוויזארד

1. עבור לעמוד **AI Analysis**
2. לחץ על **"צור ניתוח חדש"** בכותרת העמוד
3. הוויזארד ייפתח במודל

### שלב 1: בחירת מנוע, שפה ותבנית

1. **בחר מנוע AI** (Gemini או Perplexity) - שדה חובה מסומן ב-*
2. **בחר שפת משוב** - עברית או English
3. **בחר תבנית ניתוח** מתוך 4 התבניות הזמינות
4. לחץ על **"המשך לשלב 2"**

> **טיפ:** Gemini מומלץ למתחילים. ודא שהגדרת מפתח API בפרופיל המשתמש.

### שלב 2: הגדרת פילטרים ומאפיינים

השלב מחולק לשני סקשנים שניתן לפתוח/לסגור (אקורדיאון):

#### פילטרים (אם קיימים)

- חשבון מסחר (אם נדרש)
- פילטרים נוספים לפי התבנית

#### מאפיינים לניתוח

- מלא את כל השדות הנדרשים (מסומנים ב-*)
- עבור שדות תאריך, השתמש בלוח שנה לבחירת טווח
- עבור טיקרים, בחר מהרשימה או הזן "אחר"
- עבור תבנית 2 (טכני), תוכל לבחור אינדיקטורים מתוך רשימת שיטות המסחר
- עבור תבנית 3 (ביצועים), תוכל לבחור טווח תאריכים

לחץ על **"המשך לשלב 3"** לאחר מילוי כל השדות הנדרשים.

> **שימו לב:** המערכת תוודא שכל השדות הנדרשים מולאו ותציג הודעות שגיאה ברורות אם משהו חסר.

### שלב 3: תוצאות הניתוח

1. המערכת תציג הודעת טעינה
2. הניתוח ייווצר תוך מספר שניות
3. התוצאות יוצגו בפורמט Markdown עם אינפוגרפיקות
4. תוכל לבצע פעולות:
   - **שמור כהערה** - שמירת התוצאות כהערה מקושרת
   - **ייצא** - ייצוא ל-PDF, Markdown או HTML
   - **חזור** - חזרה לשלב 2 לעריכה

### הרצה חוזרת של ניתוח

1. מהיסטוריית הניתוחים, לחץ על **"הרץ שוב"**
2. הוויזארד ייפתח ישירות לשלב 3 עם התוצאות הקיימות
3. תוכל לשמור או לייצא את התוצאות מחדש

## 📈 הצגת תוצאות

### פורמט התוצאות

התוצאות מוצגות בפורמט Markdown עם:

- **כותרות** - ארגון ברור של המידע
- **רשימות** - נקודות עיקריות
- **טבלאות** - נתונים מובנים
- **גרפים** - אינפוגרפיקות (אם רלוונטי)

### אינפוגרפיקות

המערכת מזהה אוטומטית מתי להציג גרפים:

- נתונים מספריים - גרפי עמודות
- מגמות זמן - גרפי קו
- השוואות - גרפי עוגה

### פעולות על תוצאות

- **שמירה** - הניתוח נשמר אוטומטית להיסטוריה
- **הדפסה** - לחץ על כפתור "הדפס"
- **ייצוא** - ייצוא ל-PDF או Markdown

## ❓ שאלות נפוצות

### איזה מנוע AI מומלץ

**Gemini** מומלץ למתחילים - הוא חינמי, יציב וקל לשימוש. **Perplexity** מתאים לניתוחים הדורשים חיפוש אינטרנט.

### כמה עולה שימוש במערכת

המערכת עצמה חינמית. עלויות תלויות במנוע AI שבו תבחר:

- **Gemini** - חינמי עם quotas יומיים
- **Perplexity** - Pay-per-use (בדוק באתר שלהם)

### כמה זמן לוקח ליצור ניתוח

בדרך כלל 5-15 שניות, תלוי במנוע AI ובמורכבות הניתוח.

### האם הניתוחים נשמרים

כן, כל הניתוחים נשמרים אוטומטית להיסטוריה. תוכל לצפות בהם בכל עת.

### האם אפשר לערוך תבניות

כרגע לא, אבל התבניות מתוכננות היטב ומכסות את רוב הצרכים. בעתיד נוסיף אפשרות לעריכה.

### מה לעשות אם הניתוח נכשל

1. **בדוק את הודעת השגיאה** - המערכת מציגה הודעות מפורטות:
   - אם חרגת ממכסה (quota) - תראה מתי המכסה מתאפסת ואיזה מנוע AI הגיע למכסה
   - אם מפתח API חסר - תראה הודעה ברורה עם קישור להגדרות
2. **בדוק שה-API key תקין** בהגדרות המשתמש
3. **בדוק שיש חיבור לאינטרנט**
4. **נסה שוב** - המערכת תשמור את כל הערכים שהזנת
5. אם הבעיה נמשכת, פנה לתמיכה
