## 🔄 זרימת משתמש

### תרחיש 1: ניתוח לפי סוגי השקעות

1. משתמש נכנס לעמוד "ניתוח השוואתי"
2. בוחר תקופה (למשל: חודש)
3. בוחר סוגי השקעות (swing, day, long-term)
4. לוחץ על "החל פילטרים"
5. רואה:
   - טבלת השוואות
   - גרפים השוואתיים
   - Heatmap
6. מזהה איזה סוג השקעה יותר רווחי

### תרחיש 2: ניתוח לפי שיטות מסחר

1. משתמש נכנס לעמוד "ניתוח השוואתי"
2. בוחר תקופה
3. בוחר שיטות מסחר (Moving Average, RSI)
4. לוחץ על "החל פילטרים"
5. רואה השוואה בין השיטות
6. מזהה איזה שיטה יותר אפקטיבית

## 📊 דרישות נתונים

### נתונים נדרשים מהממשק:

#### 1. נתוני תקופה
- `start_date` - תאריך התחלה
- `end_date` - תאריך סיום

#### 2. נתוני פילטרים
- `investment_types` - סוגי השקעות (array)
- `trading_methods` - שיטות מסחר (array)
- `with_plan` - עם תוכנית (boolean)
- `without_plan` - בלי תוכנית (boolean)
- `time_to_entry` - משך זמן מתכנון לכניסה (string)

#### 3. נתוני השוואות
- `comparisons` - רשימת השוואות (array):
  - `category` - קטגוריה
  - `category_type` - סוג קטגוריה (investment_type/trading_method/plan_status/time_to_entry)
  - `trades_count` - מספר טריידים
  - `average_pl` - P/L ממוצע
  - `total_pl` - P/L כולל
  - `success_rate` - אחוז הצלחה
  - `average_pl_per_trade` - P/L ממוצע לטרייד
  - `average_duration` - משך זמן ממוצע

#### 4. נתוני סיכום
- `total_categories` - סה"כ קטגוריות
- `total_trades` - סה"כ טריידים
- `total_pl` - P/L כולל

## 🔌 דרישות API

### API Endpoints נדרשים:

#### 1. GET /api/analysis/compare
**מטרה:** השוואה כללית לפי פילטרים

**Parameters:**
- `start_date` (query) - תאריך התחלה (YYYY-MM-DD)
- `end_date` (query) - תאריך סיום (YYYY-MM-DD)
- `investment_types` (query) - סוגי השקעות (comma-separated)
- `trading_methods` (query) - שיטות מסחר (comma-separated)
- `with_plan` (query) - עם תוכנית (true/false)
- `without_plan` (query) - בלי תוכנית (true/false)
- `time_to_entry` (query) - משך זמן מתכנון לכניסה

**Response:**
```json
{
  "start_date": "2025-01-01",
  "end_date": "2025-01-31",
  "filters": {
    "investment_types": ["swing", "day", "long-term"],
    "trading_methods": ["Moving Average", "RSI"],
    "with_plan": true,
    "without_plan": false,
    "time_to_entry": "1-7 days"
  },
  "comparisons": [
    {
      "category": "swing",
      "category_type": "investment_type",
      "trades_count": 25,
      "average_pl": 500,
      "total_pl": 12500,
      "success_rate": 68,
      "average_pl_per_trade": 500,
      "average_duration": 14
    }
  ],
  "summary": {
    "total_categories": 7,
    "total_trades": 50,
    "total_pl": 25000
  }
}
```
