## 🔄 זרימת משתמש

### תרחיש 1: בדיקת מצב תיק בתאריך מסוים

1. משתמש נכנס לעמוד "מצב תיק היסטורי"
2. בוחר תאריך (למשל: 15 בינואר)
3. רואה:
   - כרטיסי סיכום
   - טבלת פוזיציות
   - גרף שווי תיק
   - גרף P/L
4. לוחץ על תקופה בגרף (למשל: חודש)
5. רואה מגמה של חודש

### תרחיש 2: השוואה בין תאריכים

1. משתמש נכנס לעמוד "מצב תיק היסטורי"
2. בוחר תאריך ראשון
3. רואה את הנתונים
4. לוחץ על "השווה"
5. בוחר תאריך שני
6. רואה השוואה בין שני התאריכים

## 📊 דרישות נתונים

### נתונים נדרשים מהממשק:

#### 1. נתוני תאריך
- `date` - תאריך נבחר

#### 2. נתוני סיכום
- `total_cash_balance` - יתרת מזומן כוללת
- `cash_balance_by_account` - יתרות לפי חשבון (array)
- `total_portfolio_value` - שווי תיק כולל
- `total_realized_pl` - P/L ממומש כולל
- `total_unrealized_pl` - P/L לא ממומש כולל
- `total_pl` - P/L כולל
- `open_positions_count` - מספר פוזיציות פתוחות
- `positions_count_by_account` - פוזיציות לפי חשבון (array)

#### 3. נתוני פוזיציות
- `positions` - רשימת פוזיציות (array):
  - `trading_account_id`
  - `trading_account_name`
  - `ticker_id`
  - `ticker_symbol`
  - `quantity`
  - `direction` (long/short)
  - `average_price`
  - `market_value`
  - `realized_pl`
  - `unrealized_pl`
  - `total_pl`
  - `pl_percentage`

#### 4. נתוני שווי תיק לאורך זמן
- `portfolio_value_history` - היסטוריית שווי תיק (array):
  - `date`
  - `portfolio_value`

#### 5. נתוני P/L לאורך זמן
- `pl_history` - היסטוריית P/L (array):
  - `date`
  - `realized_pl`
  - `unrealized_pl`
  - `total_pl`

#### 6. נתוני השוואה (אם נבחר תאריך שני)
- `comparison` - נתוני השוואה:
  - `date1` - תאריך ראשון
  - `date2` - תאריך שני
  - `cash_balance_diff` - שינוי ביתרות
  - `portfolio_value_diff` - שינוי בשווי תיק
  - `pl_diff` - שינוי ב-P/L
  - `positions_diff` - שינוי במספר פוזיציות

```json
    "total_realized_pl": 10000,

    "open_positions_count": 5,
    "positions_count_by_account": [
      {"account_id": 1, "count": 3},
      {"account_id": 2, "count": 2}
    ]
  },
  "positions": [
    {
      "trading_account_id": 1,
      "trading_account_name": "Account #1",
      "ticker_id": 1,
```

#### 2. GET /api/portfolio/positions/{date}
**מטרה:** קבלת פוזיציות בלבד בתאריך מסוים

**Parameters:**
- `date` (path) - תאריך (YYYY-MM-DD)

**Response:**
```json
{
  "date": "2025-01-15",
  "positions": [...]
}
```

#### 3. GET /api/portfolio/comparison
**מטרה:** השוואה בין שני תאריכים

**Parameters:**
- `date1` (query) - תאריך ראשון (YYYY-MM-DD)
- `date2` (query) - תאריך שני (YYYY-MM-DD)

**Response:**
```json
{
  "date1": "2025-01-10",
  "date2": "2025-01-15",
  "comparison": {

    },

      "diff": 5000,
      "diff_percentage": 7.14
    },

      "date1": 22000,

    },
    "positions_count": {
```
