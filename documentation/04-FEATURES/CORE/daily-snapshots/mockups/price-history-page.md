4. רואה:
   - גרף מחירים

   - אינדיקטורים

6. רואה מגמה ארוכה יותר

1. משתמש נכנס לעמוד "היסטוריית מחירים"
2. בוחר טיקר ראשון (AAPL)
3. בוחר טיקרים נוספים (TSLA, MSFT)

## 📊 דרישות נתונים

### נתונים נדרשים מהממשק:

#### 1. נתוני טיקר
- `ticker_id` - מזהה טיקר
- `ticker_symbol` - שם טיקר

#### 2. נתוני תקופה
- `start_date` - תאריך התחלה
- `end_date` - תאריך סיום

#### 3. נתוני מחירים יומיים
- `price_history` - היסטוריית מחירים (array):
  - `date`
  - `price`
  - `change_percentage`
  - `change_amount`
  - `volume` (אם זמין)
  - `high` (אם זמין)
  - `low` (אם זמין)
  - `open` (אם זמין)

#### 4. נתוני אינדיקטורים
- `indicators` - אינדיקטורים:
  - `daily_change` - שינוי יומי (אחוז וסכום)
  - `weekly_change` - שינוי שבועי (אחוז וסכום)
  - `monthly_change` - שינוי חודשי (אחוז וסכום)
  - `yearly_change` - שינוי שנתי (אחוז וסכום)

#### 5. נתוני השוואה (אם נבחרו מספר טיקרים)
- `comparison` - נתוני השוואה:
  - `tickers` - רשימת טיקרים (array)
  - `price_history_by_ticker` - היסטוריית מחירים לפי טיקר (object)

## 🔌 דרישות API

### API Endpoints נדרשים:

#### 1. GET /api/tickers/{ticker_id}/price-history
**מטרה:** קבלת היסטוריית מחירים של טיקר

**Parameters:**
- `ticker_id` (path) - מזהה טיקר
- `start_date` (query) - תאריך התחלה (YYYY-MM-DD)
- `end_date` (query) - תאריך סיום (YYYY-MM-DD)

**Response:**
```json
{
  "ticker_id": 1,
  "ticker_symbol": "AAPL",
  "start_date": "2025-01-01",
  "end_date": "2025-01-31",
  "price_history": [
    {
      "date": "2025-01-01",
      "price": 150.00,
      "change_percentage": null,
      "change_amount": null,
      "volume": 1200000,
      "high": 152.00,
      "low": 149.00,
      "open": 150.50
    },
    {
      "date": "2025-01-02",
      "price": 151.00,
      "change_percentage": 0.67,
      "change_amount": 1.00,
      "volume": 1500000,
      "high": 152.00,
      "low": 150.00,
      "open": 150.50
    }
  ],
  "indicators": {
    "daily_change": {
      "percentage": 0.65,
      "amount": 1.00
    },
    "weekly_change": {
      "percentage": 3.33,
      "amount": 5.00

      "percentage": 12.5,
      "amount": 18.75
    },
    "yearly_change": {
      "percentage": 25.0,
      "amount": 37.50
    }
  }
}
```

#### 2. GET /api/tickers/price-comparison
**מטרה:** השוואה בין מספר טיקרים

**Parameters:**
- `ticker_ids` (query) - רשימת מזההי טיקרים (comma-separated)
- `start_date` (query) - תאריך התחלה (YYYY-MM-DD)
- `end_date` (query) - תאריך סיום (YYYY-MM-DD)

**Response:**
```json
{
  "start_date": "2025-01-01",
  "end_date": "2025-01-31",
  "tickers": [
    {"id": 1, "symbol": "AAPL"},
    {"id": 2, "symbol": "TSLA"},
    {"id": 3, "symbol": "MSFT"}
  ],
  "price_history_by_ticker": {

    "3": [
      {"date": "2025-01-01", "price": 300.00},
```
