## שלבים לבדיקה מלאה

התהליך המלא שצריך לבדוק:

1. **הרצת ניתוח חדש** (או rerun של ניתוח קיים)
   - להריץ ניתוח עם תבנית וטיקר
   - לחכות לסיום הניתוח
   - לוודא שתוצאות נשמרו במטמון

2. **פתיחת מסך תוצאות**
   - לפתוח את מסך התוצאות מהטבלה או ישירות
   - לוודא שהתוצאות מוצגות

3. **לחיצה על "שמור כהערה"**
   - לוודא שהכפתור מחובר (setupResultsModalButtons)
   - לוודא שיש לוגים בקונסול

4. **בחירת סוג אובייקט**
   - לוודא שמודול בחירת סוג נפתח
   - לבחור "טיקר"
   - לוודא שמודול בחירת טיקר נפתח

5. **בחירת טיקר**
   - לוודא שהרשימה נטענת
   - לבחור טיקר
   - לוודא שמודול הערות נפתח

6. **שמירת הערה**
   - לוודא שהתוכן מוצג נכון
   - לוודא שסוג האובייקט והטיקר נבחרים
   - לשמור את ההערה
   - לוודא שההערה נשמרת

7. **בדיקה בעמוד הערות**
   - לנווט לעמוד הערות
   - לוודא שההערה מוצגת
   - לוודא שהתוכן נכון
   - לוודא שהטיקר משויך נכון
