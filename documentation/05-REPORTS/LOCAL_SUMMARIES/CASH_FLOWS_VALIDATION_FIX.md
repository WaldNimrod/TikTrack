# תיקון ולידציה - Cash Flows

## הבעיה
כפתור שמירה לא עובד בעמוד Cash Flows - לא קורה כלום

## סיבה
הייתה **שגיאת תחביר בקובץ הקונפיגורציה**:

```javascript
const cashFlowModalConfig = {
    id: 'cashFlowModal',
    entityType: 'cash_flow'  // ❌ חסר פסיק כאן!
    title: {
        add: 'הוספת תזרים מזומנים',
        edit: 'עריכת תזרים מזומנים'
    },
```

השגיאה גרמה ל:
- הקובץ לא נטען
- המודל לא נוצר
- הכפתור לא עבד

## תיקון
```javascript
const cashFlowModalConfig = {
    id: 'cashFlowModal',
    entityType: 'cash_flow',  // ✅ הוספתי פסיק
    title: {
        add: 'הוספת תזרים מזומנים',
        edit: 'עריכת תזרים מזומנים'
    },
```

## איך זה עובד עכשיו
1. ✅ הקובץ נטען נכון
2. ✅ המודל נוצר
3. ✅ הכפתור עובד
4. ✅ הולידציה פועלת (מתוך saveCashFlow)

## בדיקה
כעת צריך לבדוק:
1. פתיחת עמוד Cash Flows
2. לחיצה על "הוסף תזרים מזומנים"
3. ניסיון שמירה ללא נתונים
4. וידוא הודעות שגיאה מופיעות
