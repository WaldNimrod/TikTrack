## 🔧 הוראות בדיקה

### הגדרת סביבה:
1. וודא שהשרת רץ: `./start_server.sh`
2. פתח דפדפן: `http://127.0.0.1:8080/preferences.html`
3. פתח Developer Tools (F12)
4. עבור ל-Console tab
5. עבור ל-Network tab לבדיקות API

### בדיקות API ידניות:
```bash
# קבלת פרופילים
curl 'http://127.0.0.1:8080/api/preferences/profiles?user_id=1' | python3 -m json.tool

# הפעלת פרופיל
curl -X POST 'http://127.0.0.1:8080/api/preferences/profiles/activate' \
  -H 'Content-Type: application/json' \
  -d '{"user_id": 1, "profile_id": 2}' | python3 -m json.tool

# קבלת העדפות
curl 'http://127.0.0.1:8080/api/preferences/user?user_id=1&profile_id=2' | python3 -m json.tool

# קבלת העדפה בודדת
curl 'http://127.0.0.1:8080/api/preferences/user/single?user_id=1&preference_name=primaryCurrency&profile_id=0' | python3 -m json.tool
```

### בדיקות Console:
```javascript
// בדיקת מערכות
console.log('PreferencesCore:', window.PreferencesCore);
console.log('ProfileManager:', window.ProfileManager);
console.log('PreferencesUI:', window.PreferencesUI);

// בדיקת החלפת פרופיל
window.switchProfile(2, 1).then(r => console.log('Switch result:', r));

// בדיקת קבלת פרופילים
```
