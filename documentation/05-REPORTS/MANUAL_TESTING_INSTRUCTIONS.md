## 📋 רשימת בדיקות

### 1. בדיקת עמודים מרכזיים

#### עמודים לבדיקה:
- ✅ `index.html` - דף ראשי
- ✅ `trades.html` - עמוד עסקאות
- ✅ `executions.html` - עמוד ביצועים
- ✅ `preferences.html` - עמוד העדפות
- ✅ `alerts.html` - עמוד התראות
- ✅ `tickers.html` - עמוד טיקרים
- ✅ `notes.html` - עמוד הערות

#### מה לבדוק:
1. **Header נטען:**
   - וידוא שה-header מופיע בראש העמוד
   - בדיקת תפריט ניווט
   - בדיקת כפתורי משתמש

2. **קונסולה נקייה:**
   - פתיחת DevTools (F12)
   - בדיקת Console tab
   - וידוא שאין שגיאות קריטיות
   - בדיקת אזהרות

3. **פונקציונליות בסיסית:**
   - בדיקת כפתורים עובדים
   - בדיקת טבלאות נטענות
   - בדיקת מודלים נפתחים

### 2. בדיקת ביצועים

#### Network Tab:
1. פתיחת DevTools → Network tab
2. רענון העמוד (Ctrl+R / Cmd+R)
3. בדיקת:
   - זמן טעינה כולל
   - זמן טעינת `init-system` package
   - זמן טעינת `base` package
   - גודל קבצים

#### Performance Tab:
1. פתיחת DevTools → Performance tab
2. התחלת הקלטה
3. רענון העמוד
4. עצירת הקלטה
5. בדיקת:
   - זמן איתחול
   - זמן טעינת scripts
   - זמן ביצוע initialization

### 3. בדיקת Edge Cases

#### עמודים מיוחדים:

- `tooltip-editor.html` - עורך tooltips
- עמודי auth (login, register) - וידוא שה-header לא נטען

#### מה לבדוק:

2. בדיקת קונסולה (אין שגיאות)
3. בדיקת פונקציונליות ספציפית

### 4. בדיקת סדר טעינה

#### Console Logs:
1. פתיחת DevTools → Console tab
2. רענון העמוד
3. בדיקת הודעות:
   - `Loading Init System...`
   - `Starting Unified App Initialization`
   - `Header System initialized`
   - `Unified App Initialization completed`

#### Network Tab - סדר טעינה:
1. פתיחת DevTools → Network tab
2. רענון העמוד
3. בדיקת סדר טעינה:
   - `base` package נטען ראשון
   - כל ה-packages נטענים אחר כך
   - `init-system` package נטען אחרון
