- הסקריפט משתמש בשמות טבלאות ו-ORM objects במקום שמות עמודות ספציפיים
- השימושים היחידים בשמות עמודות הם:

- כל שאר המחיקות הן כלליות (DELETE FROM table) ללא התניה על שמות עמודות

- המודלים צריכים להתאים למבנה החדש
