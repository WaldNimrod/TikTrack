## 📋 הוראות הרצה

### הרצת הסקריפט:
1. פתח את הדפדפן ונווט לכל עמוד במערכת
2. פתח את הקונסולה (F12)
3. הרץ: `await window.runPreferencesPagesScan()`
4. או ייבא את הקובץ: `<script src="scripts/preferences-pages-scanner.js"></script>`

### ייצוא התוצאות:
לאחר הרצת הסקריפט, הרץ:
```javascript
const scanner = new PreferencesPagesScanner();
await scanner.scanAllPages();
scanner.exportReportAsJSON();
```
