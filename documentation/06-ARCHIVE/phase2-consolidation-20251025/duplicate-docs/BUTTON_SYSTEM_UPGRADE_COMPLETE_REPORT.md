## הוראות לתחזוקה

### תחזוקה שוטפת
1. **בדיקת כפתורים שבורים:**
   ```bash
   python3 scripts/button_system_tests.py
   ```

2. **אופטימיזציה תקופתית:**
   ```bash
   python3 scripts/optimize_button_system.py
   ```

3. **בדיקת נגישות:**
   - הפעלת screen reader
   - בדיקת מקשי קיצור
   - בדיקת aria-labels

### הוספת כפתור חדש
1. הוספה ל-`BUTTON_ICONS`
2. הוספה ל-`BUTTON_TEXTS`
3. הוספה ל-`getButtonClass()`
4. עדכון התיעוד
5. הוספת בדיקות

### עדכון כפתור קיים
1. שינוי ב-`getButtonClass()` (לכל הכפתורים מהסוג)
2. שימוש ב-`data-classes` (לכפתור ספציפי)
3. עדכון התיעוד
4. בדיקת תאימות

### ניקוי תקופתי
1. ניקוי cache:
   ```javascript
   window.advancedButtonSystem.cache.clear();
   ```

2. ניקוי לוגים:
   ```javascript
   window.advancedButtonSystem.logger.logs = [];
   ```

3. ניקוי מעקבים:
   ```javascript
   window.advancedButtonSystem.cleanup();
   ```
