## מבוא

מערכת הכפתורים המרכזית של TikTrack מספקת חוויית משתמש אחידה ונגישה בכל האתר. המדריך הזה יעזור לך להבין איך להשתמש בכפתורים השונים ולנצל את כל התכונות הזמינות.

### מה תמצא במדריך הזה?

- הסבר על כל סוגי הכפתורים
- הוראות שימוש מפורטות
- טיפים לנגישות
- פתרונות לבעיות נפוצות
- טריקים מתקדמים

## איך להשתמש בכפתורים

### לחיצה על כפתור

1. **לחיצה רגילה:** לחץ על הכפתור עם העכבר
2. **לחיצה עם מקלדת:** השתמש בטאב כדי לנווט לכפתור, ואז לחץ Enter או רווח
3. **לחיצה במובייל:** נגע בכפתור על המסך

### הבנת צבעי הכפתורים

- **🔴 אדום:** פעולות מסוכנות (מחיקה, דחייה, עצירה)
- **🟢 ירוק:** פעולות חיוביות (הוספה, שמירה, אישור, הפעלה)
- **🔵 כחול:** פעולות ראשיות (שמירה, קישור, צפייה)
- **🟡 צהוב:** פעולות אזהרה (השהיה, ארכוב, פילטר)
- **⚪ אפור:** פעולות נייטרליות (עריכה, ביטול, סגירה)

### אינדיקטורים חזותיים

- **איקונים:** כל כפתור כולל איקון שמסביר את הפעולה
- **טקסט:** רוב הכפתורים כוללים טקסט בעברית
- **צבעים:** צבעי הכפתורים עוזרים להבין את סוג הפעולה
- **גדלים:** כפתורים יכולים להיות בגדלים שונים (קטן, רגיל, גדול)

## נגישות ומקשי קיצור

### תמיכה ב-Screen Readers

כל הכפתורים במערכת כוללים:
- **aria-label:** תיאור מילולי של הפעולה
- **title:** טיפ שמסביר את הפעולה
- **role:** הגדרת תפקיד הכפתור

### מקשי קיצור

#### מקשי קיצור כלליים
- **Tab:** ניווט בין כפתורים
- **Enter/Space:** הפעלת הכפתור הנבחר
- **Escape:** סגירת מודלים וחלונות

#### מקשי קיצור ספציפיים
- **Ctrl+S:** שמירה מהירה
- **Ctrl+N:** הוספה מהירה
- **Ctrl+E:** עריכה מהירה
- **Ctrl+D:** מחיקה מהירה
- **F5:** רענון נתונים

### ניווט עם מקלדת

1. **Tab:** עבור לכפתור הבא
2. **Shift+Tab:** עבור לכפתור הקודם
3. **Enter/Space:** הפעל את הכפתור הנבחר
4. **Arrow Keys:** ניווט בתוך קבוצות כפתורים

## התאמה אישית

### שינוי הגדרות נגישות

```javascript
// הפעלת הגדרות נגישות מתקדמות
window.advancedButtonSystem.config.accessibility = {
    highContrast: true,
    largeText: true,
    reducedMotion: true
};
```

### התאמת צבעים

```javascript
// שינוי צבעי כפתורים
document.documentElement.style.setProperty('--btn-primary-color', '#your-color');
document.documentElement.style.setProperty('--btn-danger-color', '#your-danger-color');
```

### הגדרת מקשי קיצור מותאמים

```javascript
// הוספת מקש קיצור מותאם
document.addEventListener('keydown', function(e) {
    if (e.ctrlKey && e.key === 'y') {
        // הפעלת פעולה מותאמת
        e.preventDefault();
        // קוד הפעולה
    }
});
```

## פתרון בעיות

### בעיות נפוצות

#### הכפתור לא מגיב ללחיצה
**סיבות אפשריות:**
- הכפתור לא נטען כראוי
- יש שגיאה JavaScript
- הפונקציה לא מוגדרת

**פתרונות:**
1. רענן את הדף
2. בדוק את הקונסול לשגיאות
3. נסה כפתור אחר מאותו סוג

#### הכפתור לא נראה נכון
**סיבות אפשריות:**
- CSS לא נטען
- יש התנגשות עיצובית
- הגדרות נגישות משפיעות

**פתרונות:**
1. בדוק שהקבצים נטענו
2. נסה דפדפן אחר
3. בדוק הגדרות נגישות

#### הכפתור לא נגיש
**סיבות אפשריות:**
- חסרים aria-labels
- אין תמיכה ב-screen reader
- הגדרות נגישות לא מופעלות

**פתרונות:**
1. הפעל screen reader
2. בדוק הגדרות נגישות
3. פנה לתמיכה טכנית

### דיווח על בעיות

כשאתה נתקל בבעיה:

1. **תעד את הבעיה:**
   - איזה כפתור לא עובד
   - מה קרה כשניסית להשתמש בו
   - איזה דפדפן אתה משתמש

2. **בדוק הגדרות:**
   - האם JavaScript מופעל
   - האם יש חסימות popup
   - האם יש תוספים שמפריעים

3. **פנה לתמיכה:**
   - שלח דוח מפורט
   - כלול צילומי מסך
   - ציין את הגרסה של הדפדפן

## טיפים וטריקים

### שימוש יעיל בכפתורים

#### 1. הבנת הקשר
- כפתורים באותו אזור עושים פעולות דומות
- צבעי הכפתורים עוזרים להבין את הפעולה
- איקונים מספקים רמזים חזותיים

#### 2. שימוש במקשי קיצור
- למד את מקשי הקיצור הנפוצים
- השתמש ב-Tab לניווט מהיר
- השתמש ב-Enter/Space להפעלה

#### 3. הבנת הודעות
- קרא הודעות שגיאה בעיון
- הבן את ההשלכות של כל פעולה
- השתמש בהודעות אישור

### טיפים למובייל

#### 1. שימוש באצבע
- נגע בכפתור במרכז
- המתן לתגובה לפני נגיעה נוספת
- השתמש בתנועות טבעיות

#### 2. ניווט
- השתמש בטאבים לניווט
- השתמש בגלילה לניווט ארוך
- השתמש ב-pinch לזום

#### 3. ביצועים
- סגור אפליקציות מיותרות
- השתמש בחיבור WiFi יציב
- רענן את הדף אם יש בעיות

### טיפים למשתמשים מתקדמים

#### 1. התאמה אישית
- הגדר מקשי קיצור מותאמים
- שנה צבעים לפי העדפות
- הפעל הגדרות נגישות מתקדמות

#### 2. אוטומציה
- השתמש במקשי קיצור לפעולות חוזרות
- הגדר פעולות אוטומטיות
- השתמש בסקריפטים מותאמים

#### 3. ביצועים
- השתמש ב-cache לדפים נפוצים
- סגור טאבים מיותרים
- נקה cache באופן קבוע
