## 🚀 הרצת הסקריפט

### דרישות מוקדמות

1. PostgreSQL צריך להיות פעיל
2. משתני סביבה של PostgreSQL מוגדרים (או להריץ דרך `start_server.sh`)

### הרצה

```bash
# דרך 1: עם משתני סביבה ידניים
export POSTGRES_HOST=localhost
export POSTGRES_DB=TikTrack-db-development
export POSTGRES_USER=TikTrakDBAdmin
export POSTGRES_PASSWORD="BigMeZoo1974!?"
export POSTGRES_PORT=5432

python3 Backend/scripts/setup_initial_users.py
```

או:

```bash
# דרך 2: הסקריפט יקח את המשתנים מ-start_server.sh
# (אם השרת כבר רץ)
python3 Backend/scripts/setup_initial_users.py
```

---

## 📋 מה הסקריפט עושה

1. **תיקון Sequence**: מתקן את sequence של PostgreSQL כדי למנוע שגיאות ID
2. **בדיקת נימרוד**: בודק אם נימרוד קיים, ואם כן - מעדכן אותו למנהל ראשי
3. **יצירת נימרוד**: אם לא קיים - יוצר אותו כמנהל ראשי
4. **עדכון סיסמה**: מעדכן את סיסמת נימרוד ל-`nimw` אם צריך
5. **יצירת משתמשים נוספים**: יוצר את `admin` ו-`user` אם לא קיימים
6. **עדכון סיסמאות**: מעדכן סיסמאות למשתמשים קיימים אם צריך
7. **סיכום**: מציג רשימה של כל המשתמשים במערכת
