# 📡 פקודת משילות: סטנדרט שמות (Naming Convention) v1.3
**project_domain:** TIKTRACK
  
1. **ישויות ליבה (Plural Mandate):** השמות `trading_accounts` ו-`trades` חייבים להיות בלשון **רבים** בכל מקום (API, UI, Docs).
2. **שדות מזהה (Singular):** מזהים ייחודיים יהיו ביחיד (`user_id`, `account_id`).
3. **שדות ערך (Singular):** שדות כספיים יהיו ביחיד (`amount`, `balance`, `price`).
