# מערכת פרטי ישויות - Entity Details System

## סקירה כללית

מערכת מתקדמת להצגת פרטים מפורטים של ישויות במערכת TikTrack, כולל טיקרים, טריידים, תכניות השקעה, עסקאות, חשבונות והתראות.

## מטרת המערכת

יצירת חווית משתמש אחידה ומתקדמת להצגת פרטי ישויות עם:
- חלון קופץ אחיד לכל סוגי הישויות
- הצגת מידע מפורט וקישורים
- אינטגרציה עם מערכת הנתונים החיצוניים
- תמיכה ב-RTL ועיצוב מתקדם

## ארכיטקטורה

### Frontend Components
- **entity-details-modal.js** - מחלקה ראשית לניהול החלון הקופץ
- **entity-details-renderer.js** - רנדור תצוגת הפרטים לפי סוג ישות
- **entity-details-api.js** - טיפול בקריאות API לקבלת פרטי ישויות
- **entity-details-modal.css** - עיצוב אחיד לחלון הקופץ

### Backend Components  
- **EntityDetailsService** - שירות לקבלת פרטי ישויות מהמסד נתונים
- **API endpoints** - `/api/entity-details/<entity_type>/<entity_id>`
- **Linked Items Integration** - אינטגרציה עם מערכת הפריטים המקושרים הקיימת

### Database Integration
- שימוש במודלים הקיימים (Ticker, Trade, TradePlan, וכו')
- הרחבת מודלים עם מידע נוסף לפי הצורך
- קישור עם מערכת הנתונים החיצוניים
