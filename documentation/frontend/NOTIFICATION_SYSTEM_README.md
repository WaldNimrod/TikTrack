# מערכת ההתראות המאוחדת - TikTrack

## 📋 סקירה כללית

מערכת ההתראות המאוחדת של TikTrack היא מערכת גלובלית המרכזת את כל ההתראות בממשק המשתמש. המערכת מחליפה את כל הקריאות ל-`alert()` ו-`confirm()` המקוריות של הדפדפן במערכת מאוחדת ומתקדמת.

## 🎯 מטרות המערכת

- **מרכז התראות מאוחד** - כל ההתראות מוצגות במקום אחד
- **עקביות בממשק** - עיצוב אחיד לכל סוגי ההתראות
- **היסטוריה מתמשכת** - שמירת כל ההתראות עם אפשרות ייצוא
- **תמיכה בזמן אמת** - התראות WebSocket עם ניטור חיבור
- **אינטגרציה מלאה** - כל העמודים מחוברים למערכת

## 🏗️ ארכיטקטורת המערכת

### קבצים מרכזיים

#### 1. `notifications-center.js`
- **תפקיד**: מרכז ההתראות הראשי
- **פונקציות עיקריות**:
  - `addNotification()` - הוספת התראה חדשה
  - `updateUI()` - עדכון ממשק המשתמש
  - `loadHistory()` - טעינת היסטוריית התראות
  - `exportLogs()` - ייצוא לוגים

#### 2. `notification-system.js`
- **תפקיד**: פונקציות גלובליות להתראות
- **פונקציות עיקריות**:
  - `showSuccessNotification()` - התראת הצלחה
  - `showErrorNotification()` - התראת שגיאה
  - `showWarningNotification()` - התראת אזהרה
  - `showInfoNotification()` - התראת מידע

#### 3. `realtime-notifications-client.js`
- **תפקיד**: תמיכה ב-WebSocket
- **פונקציות עיקריות**:
  - `connect()` - התחברות לשרת
  - `disconnect()` - ניתוק מהשרת
  - `emit()` - שליחת אירועים

## 🔧 שימוש במערכת

### הוספת התראה פשוטה

```javascript
// התראת הצלחה
window.showSuccessNotification('הפעולה הושלמה בהצלחה');

// התראת שגיאה
window.showErrorNotification('שגיאה בביצוע הפעולה');

// התראת אזהרה
window.showWarningNotification('שים לב לפרטים הבאים');

// התראת מידע
window.showInfoNotification('מידע חשוב למשתמש');
```

### התראה עם כותרת

```javascript
window.showSuccessNotification('החשבון נשמר בהצלחה', 'שמירת חשבון');
window.showErrorNotification('שגיאה בשמירה', 'שמירת חשבון');
```

### בדיקת זמינות המערכת

```javascript
if (typeof window.showSuccessNotification === 'function') {
    window.showSuccessNotification('המערכת זמינה');
}
```
