## מידע על המערכת החדשה

**TikTrack** היא אפליקציית web מותאמת אישית עם הדרישות הבאות:

**טכנולוגיות:**

- Backend: Python 3.9+ עם Flask framework
- Database: PostgreSQL 15+
- Frontend: HTML/CSS/JavaScript (static files)
- Web Server: Nginx (reverse proxy)
- Process Manager: systemd או supervisor

**דרישות חומרה (מינימום):**

- CPU: 1-2 vCPU
- RAM: 2-4GB
- Storage: 20-40GB
- Bandwidth: 1TB/חודש

**דרישות תוכנה:**

- מערכת הפעלה: Ubuntu 22.04 LTS (או דומה)
- Root Access: נדרש (או לפחות custom application support)
- Python: 3.9+ עם pip
- PostgreSQL: 15+ עם psql
- Nginx: 1.18+ עם אפשרות קונפיגורציה

**הערה:** המערכת החדשה TikTrack **אינה WordPress** - היא אפליקציית Python/Flask מותאמת אישית.

## שאלות לבדיקה

### 1. תמיכה בטכנולוגיות

האם חבילת VPS Professional תומכת ב:

- Python 3.9+ - התקנה והרצה של Python applications?
- PostgreSQL 15+ - התקנה והרצה של PostgreSQL database?
- Nginx - התקנה וקונפיגורציה של Nginx?
- Custom Applications - הרצת אפליקציות מותאמות אישית?
- Root Access - גישה root או sudo לניהול מלא?

### 2. מפרטים טכניים

אנו רואים שחבילת VPS Professional כוללת:

- CPU: 2 ליבות מעבד
- RAM: 4GB
- Storage: 60GB
- Bandwidth: 4TB/חודש

**שאלות נוספות:**

- איזו מערכת הפעלה? (Ubuntu/Debian/CentOS) - איזו גרסה?
- Storage Type: SSD או HDD?
- CPU Type: איזה סוג מעבד? (Intel/AMD, איזה דגם?)
- Network: מה מהירות הרשת? (Gigabit?)

### 3. תאימות עם דרישותינו

- האם השרת עומד בכל הדרישות הטכניות?
- האם יש מגבלות שצריך לקחת בחשבון?
- האם החבילה יכולה להחזיק את כל 10 האתרים הקיימים + TikTrack?

### 4. המלצה על חבילה

- אנו שוקלים VPS Professional ($110/חודש) - האם זה מתאים?
- או שצריך Advanced/Enterprise?
- מה ההבדל בין החבילות מבחינת תמיכה ב-custom applications?

### 5. העברת אתרים

**אנו מצפים מ-uPress לבצע את העברת כל 10 האתרים הקיימים** לחבילה החדשה.

**שאלות:**

- מה התהליך? כמה זמן זה לוקח?
- האם יש downtime במהלך ההעברה?
- מה עלויות ההעברה? (אם יש)
- מתי ניתן לתאם את ההעברה?

### 6. גיבויים

אנו יודעים שיש גיבויים כלולים ומשובחים (למרות שכתוב Excluded באתר).

**שאלות:**

- מה תדירות הגיבויים? (יומיים?)
- כמה זמן retention?
- איך מתבצע שחזור?

### 7. אבטחה

- SSL: האם SSL certificate כלול? (רואים שכלול)
- Firewall: האם יש firewall מובנה? האם ניתן להגדיר rules?
- DDoS Protection: האם יש הגנה מפני DDoS?

### 8. עלויות

- עלות VPS Professional: $110 + מע"מ/חודש (₪368.5 + מע"מ) - ידוע
- עלויות נוספות: האם יש עלויות נוספות (bandwidth, storage, וכו')?
- גיבויים: כלולים (אנו יודעים מניסיון רב שנים)
- עלות העברת אתרים: מה העלות להעברת 10 האתרים הקיימים? (אם יש)
- שדרוגים: מה העלויות לשדרוג משאבים? (למשל: RAM מ-4GB ל-8GB, או מעבר ל-Advanced)

### 9. תמיכה

- תמיכה טכנית: מה זמני התמיכה? (שעות, שפות)
- SLA: מה ה-SLA המובטח? (Uptime guarantee)
- תיעוד: האם יש תיעוד טכני מפורט?
