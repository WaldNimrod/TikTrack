## סקירה כללית

מסמך זה מתעד את כל השינויים בהגדרות השרת בסביבת הפרודקשן, כולל:
- שינויים ב-`config/settings.py`
- שינויים ב-`config/logging.py`
- שינויים ב-`config/database.py`
- שינויים אחרים בהגדרות השרת

## הוראות שימוש

### עדכון אוטומטי

```bash
# הרצת סקריפט תיעוד אוטומטי
python3 scripts/production-update/document_server_changes.py
```

הסקריפט:
- מזהה שינויים בקבצי config
- מנתח שינויים בהגדרות מפתח
- מעדכן את המסמך הזה אוטומטית

### עדכון ידני

כשמבצעים שינוי ידני בהגדרות השרת, יש לתעד:

1. **תאריך השינוי**

3. **השינוי הספציפי**

5. **השפעה על המערכת**

## הערות חשובות

- **Production config files** תמיד צריכים להיות hardcoded:
  - `IS_PRODUCTION = True`
  - `PORT = 5001`
  - `DEVELOPMENT_MODE = False`
  - `CACHE_DISABLED = False`

- **כל שינוי** בהגדרות production חייב להיות מתועד כאן

- **לפני כל שינוי** - בדוק את ההשפעה על המערכת
