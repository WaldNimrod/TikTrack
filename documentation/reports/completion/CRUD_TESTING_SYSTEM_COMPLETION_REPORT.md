## מטרת הפרויקט

יצירת מערכת בדיקות CRUD מקיפה לכל העמודים במערכת TikTrack, כולל בדיקת כל הכפתורים, טפסים, מודלים ופונקציונליות במערכת.

## מה בוצע

### ✅ תכנון ואסטרטגיה
- **תוכנית בדיקות מקיפה** עבור כל 28 העמודים במערכת
- **סיווג עמודים** לפי חשיבות וסוג פונקציונליות
- **הגדרת קריטריוני הצלחה** וציון לכל עמוד
- **מדריך בדיקות מפורט** עם רשימות צ'ק

### ✅ כלי בדיקה אוטומטיים (7 כלים)

#### 🔍 כלי ניתוח HTML סטטי
1. **`static-crud-checker.py`** - בדיקת תקינות HTML וCSS
   - בדיקת 12 עמודי CRUD עיקריים
   - אימות השתלבות מערכת CSS חדשה ב-100% מהעמודים
   - זיהוי בעיות בסיסיות (DOCTYPE, title, RTL, וכו')

2. **`detailed-button-analyzer.py`** - ניתוח מפורט של כפתורים
   - זיהוי כל הכפתורים בכל עמוד
   - סיווג כפתורים לפי פונקציונליות (CRUD/Other)
   - ניתוח מודלים וטבלאות
   - חישוב ציון פונקציונליות לכל עמוד

#### 🌐 כלי בדיקה עם שרתר פעיל
3. **`auto-crud-checker.py`** - בדיקת נגישות עמודים
   - בדיקת HTTP status codes
   - בדיקת זמן תגובה
   - אימות תוכן בסיסי

4. **`test-crud-buttons.py`** - בדיקת API endpoints
   - בדיקת נגישות endpoints עבור כל עמוד
   - בדיקת תגובות API
   - אימות תקינות שרת

#### 📋 כלי ליצירת מדריכים
5. **`manual-crud-tester.py`** - בדיקות ידניות מונחות
   - יצירת רשימות בדיקה מפורטות
   - איסוף תוצאות באופן מובנה
   - דוחות JSON מפורטים

6. **`create-crud-testing-dashboard.py`** - יצירת דשבורד
   - דשבורד HTML אינטראקטיבי מלא
   - מעקב התקדמות בזמן אמת
   - ייצוא תוצאות

7. **`sample-crud-test-results.py`** - דוגמאות ודוחות
   - יצירת דוחות דוגמה
   - הדגמת פורמט תוצאות
   - הדרכה על שימוש בכלים
