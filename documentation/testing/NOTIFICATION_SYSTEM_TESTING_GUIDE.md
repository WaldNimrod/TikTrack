## 🚀 **הכנה לבדיקות**

### **דרישות מקדימות:**
1. ✅ השרת פועל על `http://localhost:8080`
2. ✅ דף העדפות נגיש
3. ✅ קונסול הדפדפן פתוח (F12)
4. ✅ מטמון הדפדפן מנוקה

### **קבצים לבדיקה:**
- `trading-ui/preferences.html` - דף העדפות
- `trading-ui/index.html` - דף הבית
- `trading-ui/trades.html` - עמוד עסקאות
- `trading-ui/accounts.html` - עמוד חשבונות
- `trading-ui/executions.html` - עמוד ביצועים

✅ **הודעה צפויה:**

2. לחץ "מחק"
3. אשר את המחיקה

✅ **הודעה צפויה:**
- "טרייד נמחק בהצלחה!" (קטגוריה: business)

## 📊 **בדיקת קטגוריות**

### **5. בדיקת כל קטגוריה בנפרד**

#### **קטגוריה: System (מערכת)**
1. לך לדף העדפות
2. בטל את "התראות מערכת"
3. לך לדף הבית ולחץ על ניקוי מטמון
4. **תוצאה צפויה:** אין הודעות

#### **קטגוריה: Business (עסקי)**
1. לך לדף העדפות
2. בטל את "התראות עסקיות"
3. לך לעמוד עסקאות ושמור טרייד
4. **תוצאה צפויה:** אין הודעות

#### **קטגוריה: UI (ממשק משתמש)**
1. לך לדף העדפות
2. בטל את "התראות ממשק משתמש"
3. לך לדף הבית ולחץ על ניקוי מטמון
4. **תוצאה צפויה:** אין הודעת "הדף ירענן..."

#### **קטגוריה: Development (פיתוח)**
1. לך לדף העדפות
2. בטל את "התראות פיתוח"
3. לך לעמוד לינטר ולחץ על "רענון רשימת קבצים"
4. **תוצאה צפויה:** אין הודעות

#### **קטגוריה: Performance (ביצועים)**
1. לך לדף העדפות
2. הפעל את "התראות ביצועים"
3. בצע פעולות שמעוררות הודעות ביצועים
4. **תוצאה צפויה:** הודעות ביצועים מופיעות

## 🐛 **דיבוג בעיות**

### **6. פתרון בעיות נפוצות**

#### **בעיה: אין הודעות מופיעות**
✅ **פתרון:**
1. בדוק בקונסול אם יש שגיאות
2. וודא שהקטגוריה מופעלת בהעדפות
3. בדוק אם הפונקציות נטענו:
```javascript
console.log(typeof window.showSuccessNotification);
console.log(typeof window.shouldShowNotification);
```

#### **בעיה: תיבות סימון קטנות/לא נראות**
✅ **פתרון:**
1. רענן את הדף (Ctrl+F5)
2. בדוק שהקובץ `_forms-advanced.css` נטען
3. בדוק בקונסול אם יש שגיאות CSS

#### **בעיה: הודעות בקטגוריה הלא נכונה**
✅ **פתרון:**
1. השתמש בקידוד מפורש במקום זיהוי אוטומטי
2. בדוק את הלוגים בקונסול
3. עדכן את אלגוריתם הזיהוי

#### **בעיה: שגיאות JavaScript**
✅ **פתרון:**
1. בדוק שהקבצים נטענו בסדר הנכון
2. בדוק אם יש שגיאות syntax
3. בדוק אם הפונקציות מוגדרות נכון

```javascript
console.timeEnd('category-detection');
```

```javascript

console.timeEnd('explicit-category');
```
