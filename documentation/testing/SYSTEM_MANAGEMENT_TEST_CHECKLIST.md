# System Management Test Checklist

## מטרת המסמך

מסמך זה מכיל checklist מפורט לבדיקות ידניות של עמוד מנהל מערכת.

## כללי

- [ ] העמוד נטען ללא שגיאות בקונסולה
- [ ] כל הסקשנים מופיעים בעמוד
- [ ] אין שגיאות JavaScript בקונסולה
- [ ] אין אזהרות קריטיות בקונסולה
- [ ] כל הכפתורים עובדים
- [ ] Auto-refresh פועל כהלכה

## 1. Dashboard Section (sm-dashboard)

### נתונים בסיסיים

- [ ] בריאות מערכת מוצגת נכון (0-100%)
- [ ] ציון מערכת נכון (0-100%)
- [ ] זמן תגובה מוצג נכון (במילישניות)
- [ ] זמן פעילות מוצג נכון
- [ ] זיכרון זמין מוצג נכון (%)
- [ ] עומס CPU מוצג נכון (%)

### מידע סביבה

- [ ] סביבה מוצגת נכון (Development/Production)
- [ ] פורט שרת מוצג נכון (8080/5001)
- [ ] שם בסיס נתונים מוצג נכון

### קישורים מהירים

- [ ] "ניהול מטמון מלא" → `/cache-management`
- [ ] "ניטור שרת מלא" → `/server-monitor`
- [ ] "דשבורד נתונים חיצוניים" → `/external-data-dashboard`

### רכיבי מערכת

- [ ] סטטוס שרת מוצג נכון
- [ ] סטטוס בסיס נתונים מוצג נכון
- [ ] סטטוס מטמון מוצג נכון
- [ ] סטטוס נתונים חיצוניים מוצג נכון

### מידע תהליכים

- [ ] זמן תגובה נכון

### מידע מערכת

- [ ] ארכיטקטורה מוצגת נכון

### קישור מהיר

- [ ] "ניטור שרת מלא" → `/server-monitor`
