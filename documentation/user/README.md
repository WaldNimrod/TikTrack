# TikTrack User Guide

## 📋 Overview

Welcome to the TikTrack Trading Management System user guide. This comprehensive guide will help you navigate and effectively use all features of the system.

## 🚀 Getting Started

### **First Time Setup**
1. **Access the System**: Open your web browser and navigate to the TikTrack application
2. **Login**: Use your provided credentials to access the system
3. **Dashboard**: Familiarize yourself with the main dashboard layout
4. **Navigation**: Learn the main navigation menu and page structure

### **System Overview**
TikTrack is a comprehensive trading management system that includes:
- **Trade Management**: Track and manage your trading activities
- **Account Management**: Monitor multiple trading accounts
- **Alert System**: Set up price and condition alerts
- **Cash Flow Tracking**: Monitor financial flows
- **Reporting**: Generate detailed reports and analytics

## 📊 Main Features

### **1. Trade Management**
- **View Trades**: Browse all your trading activities
- **Add New Trade**: Create new trade entries
- **Edit Trades**: Modify existing trade information
- **Trade Status**: Track trade status (Open, Closed, Canceled)
- **Linked Items**: Connect trades to related items

### **2. Account Management**
- **Account Overview**: View account balances and performance
- **Multiple Accounts**: Manage several trading accounts
- **Currency Support**: Handle multiple currencies
- **Account History**: Track account changes over time

### **3. Alert System**
- **Price Alerts**: Set alerts for specific price levels
- **Condition Alerts**: Create alerts based on trading conditions
- **Alert Status**: Monitor active and triggered alerts
- **Notification Settings**: Configure alert notifications

### **4. Cash Flow Tracking**
- **Income Tracking**: Record trading income
- **Expense Management**: Track trading expenses
- **Flow Categories**: Organize cash flows by category
- **Financial Reports**: Generate cash flow reports

## 🔧 Common Operations

### **Adding a New Trade**
1. Navigate to the Trades page
2. Click "Add New Trade" button
3. Fill in the required information:
   - Account selection
   - Ticker/Symbol
   - Trade type (Buy/Sell)
   - Quantity
   - Price
   - Date
4. Click "Save" to create the trade

### **Setting Up an Alert**
1. Go to the Alerts page
2. Click "Add New Alert"
3. Configure alert parameters:
   - Select ticker/symbol
   - Set price condition
   - Choose alert type
   - Set notification preferences
4. Save the alert

### **Managing Accounts**
1. Access the Accounts page
2. View account list and balances
3. Add new accounts if needed
4. Update account information
5. Monitor account performance

### **Tracking Cash Flows**
1. Navigate to Cash Flows page
2. Record income and expenses
3. Categorize transactions
4. Monitor financial trends
5. Generate reports

#### Currency Exchange Pairing
- כל זוג המרה מוצג עם תגית 🔁 בטבלה ועם הדגשת שתי השורות בעת ריחוף כדי שלא תאבדו את הצד המשלים גם אחרי סינון/מיון.
- כפתור "פתח רשומה צמודה" חדש נמצא בתפריט הפעולות ומקפיץ מיד את הרשומה האחות בחלון הפרטים (ModalManagerV2).
- חלון הפרטים מציג מודול "צמד המרה" עם שני כרטיסים מוערמים (שלילי למעלה, חיובי למטה) כולל סכומים, מטבעות וכפתורי גישה מהירים לכל צד.
- במודל העריכה מופיע אותו מודול גם בלשונית "תזרים רגיל" וגם בלשונית "המרת מטבע", כך שניתן לערוך את שני הצדדים בעקביות ולראות את הנטו והעמלה במבט אחד.
- כפתור פילטר ייעודי "המרות מטבע" (שורת הכפתורים העליונה) מציג רק רשומות שה-type שלהן הוא `currency_exchange_from`/`currency_exchange_to`, כך שאפשר לצוד המרות גם לפני שמוסיפים מטא-דאטה נוסף.
