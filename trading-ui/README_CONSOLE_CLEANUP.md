# מערכת ניקוי קונסולה - TikTrack

## 🎯 מטרה

מערכת ניקוי הקונסולה מספקת פתרון דינמי לניהול הודעות הקונסולה בכל העמודים במערכת TikTrack. המערכת מאפשרת למשתמשים להגדיר זמן ניקוי אוטומטי ולהשתמש בניקוי ידני לפי הצורך.

## ⚡ תכונות עיקריות

- **🔧 הגדרות דינמיות**: זמן ניקוי מותאם אישית (30s, 60s, 2min, 5min, 10min)
- **🚀 ביצועים מהירים**: שימוש ב-localStorage לגישה מהירה
- **🔄 ניקוי אוטומטי**: פעולה רציפה ללא התערבות
- **👆 ניקוי ידני**: כפתור לניקוי מיידי
- **⚙️ אינטגרציה מלאה**: שילוב עם מערכת העדפות

## 📋 ברירות מחדל

| הגדרה | ערך | תיאור |
|--------|-----|--------|
| **זמן ניקוי** | **60 שניות** | **ברירת מחדל חדשה** |
| אחסון | localStorage + שרת | גיבוי כפול |
| ניקוי ידני | זמין | כפתור בדף העדפות |

## 🛠️ התקנה ושימוש

### 1. הוספת הקובץ לעמוד
```html
<script src="scripts/console-cleanup.js"></script>
```

### 2. הגדרת זמן ניקוי
```javascript
// עדכון ל-60 שניות (ברירת מחדל)
await updateConsoleCleanupInterval(60000);

// עדכון ל-30 שניות
await updateConsoleCleanupInterval(30000);

// ביטול ניקוי אוטומטי
await updateConsoleCleanupInterval(0);
```

### 3. ניקוי ידני
```javascript
// ניקוי מיידי
manualConsoleCleanup();
```

## 🔧 הגדרות זמינות

| זמן | ערך (ms) | תיאור |
|-----|-----------|--------|
| 30 שניות | 30000 | ניקוי מהיר |
| **60 שניות** | **60000** | **ברירת מחדל** |
| 2 דקות | 120000 | ניקוי בינוני |
| 5 דקות | 300000 | ניקוי איטי |
| 10 דקות | 600000 | ניקוי איטי מאוד |
| אין ניקוי | 0 | כיבוי מוחלט |

## 🎮 שימוש בממשק

### הגדרת זמן ניקוי
1. גש לדף **העדפות**
2. במחלקת **הגדרות מערכת**
3. מצא **זמן ניקוי קונסולה**
4. בחר זמן מהרשימה
5. ההגדרה תיכנס לתוקף מיד

### ניקוי ידני
1. גש לדף **העדפות**
2. במחלקת **הגדרות מערכת**
3. לחץ **"נקה עכשיו"**
4. הקונסולה תתנקה מיד

## 🔄 שינויים אחרונים

### גרסה 1.1 (אוגוסט 2025)
- ✅ **שינוי ברירת מחדל**: 30 שניות → 60 שניות
- ✅ **שיפור דוקומנטציה**: הוספת פרטים מפורטים
- ✅ **אופטימיזציה**: שיפור ביצועים וטיפול בשגיאות

### גרסה 1.0 (אוגוסט 2025)
- ✅ **יצירה ראשונית**: מערכת ניקוי קונסולה בסיסית
- ✅ **אינטגרציה**: שילוב עם מערכת העדפות
- ✅ **תמיכה גלובלית**: פעולה בכל העמודים

## 🐛 פתרון בעיות

### ניקוי לא עובד
1. בדוק שהקובץ `console-cleanup.js` נטען
2. וודא שההגדרה לא מוגדרת ל-0
3. בדוק את הקונסול לשגיאות JavaScript

### הגדרות לא נשמרות
1. בדוק חיבור לשרת
2. וודא שהעדפות נשמרות ב-localStorage
3. בדוק את פונקציית `savePreference`
