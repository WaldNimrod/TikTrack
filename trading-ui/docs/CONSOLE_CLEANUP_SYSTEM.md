# מערכת ניקוי קונסולה - TikTrack

## סקירה כללית

מערכת ניקוי הקונסולה של TikTrack מספקת פתרון דינמי לניהול הודעות הקונסולה בכל העמודים במערכת. המערכת מאפשרת למשתמשים להגדיר זמן ניקוי אוטומטי ולהשתמש בניקוי ידני לפי הצורך.

## תכונות עיקריות

### 🔧 הגדרות דינמיות
- **זמן ניקוי מותאם אישית**: 30 שניות, 60 שניות, 2 דקות, 5 דקות, 10 דקות
- **ביטול ניקוי אוטומטי**: אפשרות לכיבוי מוחלט של הניקוי האוטומטי
- **ברירת מחדל**: 60 שניות (דקה אחת)

### 🚀 ביצועים
- **אתחול מהיר**: שימוש ב-localStorage לגישה מהירה להגדרות
- **ניקוי אוטומטי**: פעולה רציפה ללא צורך בהתערבות המשתמש
- **ניקוי ידני**: כפתור לניקוי מיידי בכל עת

### 🔄 אינטגרציה
- **מערכת העדפות**: שילוב מלא עם דף ההעדפות
- **שמירה אוטומטית**: עדכון מידי של הגדרות לשרת ול-localStorage
- **תמיכה גלובלית**: פעולה בכל העמודים במערכת

## קבצים רלוונטיים

### קבצים ראשיים
- `scripts/console-cleanup.js` - המערכת הראשית
- `preferences.html` - ממשק הגדרות
- `scripts/preferences.js` - ניהול העדפות
- `config/preferences.json` - הגדרות ברירת מחדל

### קבצים משולבים
כל העמודים הראשיים במערכת כוללים את הקובץ `console-cleanup.js`:

- `alerts.html`
- `notes.html`
- `planning.html`
- `tickers.html`
- `executions.html`
- `currencies.html`
- `cash_flows.html`
- `database.html`

## פונקציות API

### `initializeConsoleCleanup()`
מאתחל את מערכת ניקוי הקונסולה.

```javascript
// אתחול אוטומטי
await initializeConsoleCleanup();

// אתחול ידני
document.addEventListener('DOMContentLoaded', () => {
  initializeConsoleCleanup();
});
```

### `clearConsole(showMessage = true)`
מנקה את הקונסולה ומאתחל מחדש את הטיימר.

```javascript
// ניקוי עם הודעה
clearConsole();

// ניקוי שקט
clearConsole(false);
```

### `updateConsoleCleanupInterval(newInterval)`
מעדכן את זמן הניקוי ומאתחל מחדש את המערכת.

```javascript
// עדכון ל-60 שניות (ברירת מחדל)
await updateConsoleCleanupInterval(60000);

// עדכון ל-30 שניות
await updateConsoleCleanupInterval(30000);

// ביטול ניקוי אוטומטי
await updateConsoleCleanupInterval(0);
```

### `manualConsoleCleanup()`
ניקוי ידני מיידי של הקונסולה.

```javascript
// ניקוי ידני
manualConsoleCleanup();
```

## הגדרות זמינות

| זמן | ערך (מילישניות) | תיאור |
|-----|------------------|--------|
| 30 שניות | 30000 | ניקוי מהיר |
| **60 שניות** | **60000** | **ברירת מחדל** |
| 2 דקות | 120000 | ניקוי בינוני |
| 5 דקות | 300000 | ניקוי איטי |
| 10 דקות | 600000 | ניקוי איטי מאוד |
| אין ניקוי | 0 | כיבוי מוחלט |

## שימוש בממשק המשתמש

### הגדרת זמן ניקוי
1. גש לדף **העדפות**
2. במחלקת **הגדרות מערכת**
3. מצא את **זמן ניקוי קונסולה**
4. בחר את הזמן הרצוי מהרשימה הנפתחת
5. ההגדרה תיכנס לתוקף מיד

### ניקוי ידני
1. גש לדף **העדפות**
2. במחלקת **הגדרות מערכת**
3. לחץ על כפתור **"נקה עכשיו"**
4. הקונסולה תתנקה מיד

## ארכיטקטורה טכנית

### זרימת עבודה
1. **אתחול**: טעינת הגדרות מ-localStorage או ברירת מחדל
2. **הגדרת טיימר**: יצירת setTimeout עם הזמן המוגדר
3. **ניקוי**: ביצוע console.clear() והאתחול מחדש
4. **עדכון**: תגובה לשינויים בהגדרות

### אחסון נתונים
- **localStorage**: `tiktrack_preferences.consoleCleanupInterval`
- **שרת**: `/api/preferences/consoleCleanupInterval`
- **ברירת מחדל**: 60000 מילישניות

### ניהול מצב
- **משתנה גלובלי**: `consoleCleanupTimeout`
- **ניקוי אוטומטי**: ביטול טיימר קיים לפני יצירת חדש
- **טיפול בשגיאות**: try-catch עם הודעות אזהרה

## דוגמאות שימוש

### הגדרת ניקוי מותאם אישית
```javascript
// הגדרת ניקוי כל 2 דקות
await updateConsoleCleanupInterval(120000);
```

### בדיקת הגדרה נוכחית
```javascript
const preferences = JSON.parse(localStorage.getItem('tiktrack_preferences'));
const currentInterval = preferences?.user?.consoleCleanupInterval || 60000;
console.log(`זמן ניקוי נוכחי: ${currentInterval}ms`);
```

### ניקוי ידני עם משוב
```javascript
manualConsoleCleanup();
showNotification('הקונסולה נוקתה בהצלחה', 'success');
```

## פתרון בעיות

### ניקוי לא עובד
1. בדוק שהקובץ `console-cleanup.js` נטען
2. וודא שההגדרה לא מוגדרת ל-0
3. בדוק את הקונסול לשגיאות JavaScript

### הגדרות לא נשמרות
1. בדוק חיבור לשרת
2. וודא שהעדפות נשמרות ב-localStorage
3. בדוק את פונקציית `savePreference`

### ביצועים איטיים
1. בדוק שאין טיימרים כפולים
2. וודא שהניקוי לא מתבצע בתדירות גבוהה מדי
3. שקול להגדיל את זמן הניקוי
