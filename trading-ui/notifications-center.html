עדכון חשוב: כל הודעות השגיאה והצלחה עכשיו משתמשות במערכת החדשה עם מודל מפורט ועצירה בתהליך!

תכונות חדשות:
  • עצירה בתהליך עד לסגירה
  • כפתור העתקה ללוח עם פרטים מפורטים

דוגמאות שימוש:
// שגיאה רגילה (עכשיו עם מודל!)

// שגיאה פשוטה (legacy mode)
showSimpleErrorNotification('שגיאה', 'הודעה');

// הצלחה סופית (עם מודל!)
showFinalSuccessNotification('הצלחה', 'הודעה', {
  operation: 'save-data',
  itemsProcessed: 42
});

// שגיאה מפורטת
showCriticalErrorNotification('כותרת', 'הודעה', {
  source: 'my-function',
  function: 'processData',
  line: '123'
});
