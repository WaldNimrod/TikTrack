# Active Alerts Component
# קומפוננטת התראות פעילות

## 📋 **תיאור**

קומפוננטה מודולרית להצגת התראות פעילות (`is_triggered = 'new'`) שניתן לשלב בכל דף באפליקציה.

## 🎯 **תכונות**

- ✅ **טעינה אוטומטית** של התראות פעילות מה-API
- ✅ **כפתור "קראתי"** עם עדכון בזמן אמת
- ✅ **רענון אוטומטי** אחרי סימון כנקראה
- ✅ **אנימציות חלקות** בהסרת כרטיסיות
- ✅ **הודעות הצלחה ושגיאה** למשתמש
- ✅ **עיצוב רספונסיבי** לכל המסכים
- ✅ **Shadow DOM** - בידוד סגנונות
- ✅ **Web Component** - מודולריות מלאה

## 🚀 **שימוש**

### **1. הוספת הסקריפט לדף:**

## 🔧 **API**

### **Endpoints נדרשים:**
- `GET /api/v1/alerts/unread` - קבלת התראות פעילות
- `PATCH /api/v1/alerts/{id}/mark-read` - סימון כנקראה

### **מבנה נתונים צפוי:**
```json
{
  "status": "success",
  "data": [
    {
      "id": 1,
      "message": "התראה על מחיר",
      "condition": "מחיר > 100$",
      "type": "price_alert",
      "related_type": "ticker",
      "created_at": "2025-08-18 20:16:00",
      "is_triggered": "new"
    }
  ]
}
```
