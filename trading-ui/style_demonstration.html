ארכיטקטורת ITCSS (Inverted Triangle CSS)

המערכת שלנו משתמשת בארכיטקטורת ITCSS המאורגנת ב-6 שכבות:

  • 01-settings: משתנים, צבעים, רווחים, טיפוגרפיה
  • 02-tools: פונקציות ו-mixins (ריק)
  • 03-generic: Reset ו-base styles
  • 04-elements: אלמנטים בסיסיים (כותרות, קישורים, טפסים, כפתורים)
  • 05-objects: אובייקטי פריסה (layout, grid)
  • 06-components: רכיבי UI (כפתורים מתקדמים, טבלאות, כרטיסים, מודלים)

סדר טעינה: Bootstrap → ITCSS files → Header styles
