📊 בדיקת פילטר - סקשן 1

בדיקת מערכת הפילטרים והיחסים בין הכפתורים:

  • כפתור איפוס (↻) - מחזיר לערכי ברירת מחדל מהעדפות
  • כפתור ניקוי (×) - מנקה את כל הפילטרים
  • פילטרים: סטטוס, סוג השקעה, חשבון מסחר, תאריך, חיפוש

🔧 כלי ניפוי שגיאות

כלים זמינים לבדיקה וניפוי שגיאות:

  • פונקציית window.debugHeaderFiltersState() - מציג מצב מלא של הפילטרים
  • פונקציית window.forceOpenAllHeaderMenus() - פותח את כל תפריטי הפילטר
  • כל פעולה נרשמת בקונסולה עם לוגים מפורטים
