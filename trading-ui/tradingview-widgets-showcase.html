ווידג'טים של TradingView

1. Advanced Real-Time Chart Widget (הגרף המלא) ⭐ הכי פופולרי

מה הוא עושה:

  • גרף מלא כמו באתר TradingView (candles, indicators, overlays)
  • נתונים היסטוריים מובנים (אתה לא צריך להביא)
  • שליטה על סימול, טווח זמן, אינדיקטורים בסיסיים
  • RTL עובד
  • התאמה חלקית עם פרמטרים

מגבלות:

  • אי אפשר למשוך ממנו נתונים
  • אי אפשר overlay של טריידים משלך
  • iFrame סגור
🔵 מתאים ל-TikTrack: לשימוש כגרף מרכזי טרייד

פרמטרים זמינים:

פרמטר סוג תיאור דוגמה
symbol string סמל המניה/נכס (פורמט: EXCHANGE:SYMBOL) NASDAQ:AAPL
interval string מרווח זמן (1, 5, 15, 60, D, W, M) D
timezone string אזור זמן Asia/Jerusalem
theme string ערכת נושא (light, dark) light
style string סגנון הגרף (1-9) 1
locale string שפה (he, en, es, etc.) he
toolbar_bg string צבע רקע של סרגל הכלים #f1f3f6
enable_publishing boolean האם לאפשר פרסום רעיונות false
hide_top_toolbar boolean האם להסתיר את סרגל הכלים העליון false
hide_legend boolean האם להסתיר את ה-legend false
save_image boolean האם לאפשר שמירת תמונה false
container_id string מזהה הקונטיינר tradingview_chart
autosize boolean התאמה אוטומטית לגודל true
width string רוחב הווידג'ט (אם autosize=false) 100%
height string גובה הווידג'ט 600
